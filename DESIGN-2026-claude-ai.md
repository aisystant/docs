## 1. ГЛАВНАЯ СТРАНИЦА (guide.system-school.ru/)

### Визуальный язык
- **Цветовая палитра:** 
  - Primary: Фиолетовый градиент (#5B4FE8 → #8B5CF6)
  - Accent: Яркий изумруд (#10B981)
  - Text: Глубокий чёрный (#0A0A0A)
  - Background: Белый (#FFFFFF) с микро-текстурой
  - Neutral: Серые 50-900 (системные)

- **Типография:**
  - Заголовки: Syne (bold, 800-weight), tracking -2px
  - Body: Inter (400-500), line-height 1.6
  - Code/Meta: JetBrains Mono (для примеров)

- **Спейсинг:** 8px grid, padding 40px-96px по секциям

## 2. СТРАНИЦА ПЕРСОНАЛЬНЫХ РУКОВОДСТВ (после входа)

### Навигационная структура
```
[Logo] [Dashboard] [Guides] [Settings] [Profile ▼]
```

### Layout: Two-column

#### Левая колона (240px, фиксированная)
```
📊 Dashboard
  ├─ Мой прогресс
  ├─ Мои руководства
  ├─ Сохранённые
  └─ История
```
- Background: #F9FAFB
- Links с иконками (20px)
- Active state: фиолетовый accent + bold

#### Основная колона (960px)

**Заголовок раздела**
```
Мои персональные руководства
```
- Под ним: "Адаптированы на основе вашего уровня развития"
- Фильтры: [Все] [Личное развитие] [Рабочее развитие] [Исследовательское]

**Карточки руководств (сетка 3 колонки)**

Каждая карточка:
```
┌─────────────────────┐
│ [Image/Icon Cover]  │
│                     │
│ Название            │
│ Саморазвитие        │
│                     │
│ "Шаг 1 из 7"        │

│ [Продолжить]        │

```

Design:
- Width: ~300px (3 per row на 960px)
- Card height: 380px
- Cover image: 300x150px, gradient-overlay
- Progress bar: фиолетовый, 6px высота
- Buttons: secondary (outline)

**Детальный вид одного руководства** (при клике)
```
┌─────────────────────────────────────┐
```

Design:
- Max-width: 800px
- Line-height: 1.8
- Margin notes: справа 40px, фон #F0F0F5

## 4. КАК ИСПОЛЬЗОВАТЬ claude.ai/design

1. **Откройте** https://claude.ai/design
2. **Создайте новый проект:** "System School Guide"
3. **Установите дизайн-токены** из раздела 3
4. **Спроектируйте 3 главных скрина:**
   - Hero page (guide.system-school.ru/)
   - Three tracks grid
   - Personal guide detail view
5. **Добавьте компоненты:**
   - Button (primary, secondary, outline)
   - Card
   - Navigation item
   - Progress bar
   - Input field (для фильтров)
6. **Экспортируйте** в React/HTML
