#### §2.09 Упражнения на понятия

Три упражнения на применение понятий Раздела 2. Цель — перевести «понимаю» в «применяю».

---

**Упражнение 1. Опознать мем**

Прочитайте три высказывания. Для каждого определите: какой мем за ним стоит? Откуда такой мем обычно появляется? Какая конкурентоспособная замена возможна?

> *А. «Я не буду осваивать ИИ — это очередная мода, через пару лет всё переменится».*
>
> *Б. «Учиться чему-то новому после 40 — поздно, мозг уже не тот».*
>
> *В. «У меня нет времени на ведение заметок — я лучше эти полчаса поработаю».*

Запишите для каждого: мем → источник → замена.

---

**Упражнение 2. Различить экзотело и экзокортекс**

Из списка ниже определите для каждого пункта: это компонент экзотела, экзокортекса или одновременно того и другого? Объясните, по какой функции.

1. Смартфон с приложением для заметок.
2. Кружка для кофе на рабочем столе.
3. Записная книжка в кармане.
4. Подписка на платный поиск.
5. Велосипед.
6. Папка «архив» на ноутбуке с проектами за 5 лет.
7. Очки для чтения.
8. Папка избранного в браузере с непрочитанными статьями.

---

**Упражнение 3. Найти узкое горлышко киберчеловека**

Вспомните одну важную задачу, которую вы хотите решить в ближайшие 3 месяца. Пройдите по шести киберхарактеристикам и спросите для каждой: достаточно ли её у меня для этой задачи?

| Характеристика | Достаточно для задачи? (да / частично / нет) | Что мешает, если «частично» / «нет»? |
|----------------|---------------------------------------------|--------------------------------------|
| Агентность | | |
| Интеллектуальная развитость | | |
| Стрессоустойчивость | | |
| Ресурсность | | |
| Техноинтеграция | | |
| Калибр | | |

Выделите ту, которая окажется узким горлышком именно для этой задачи. Назовите одно действие, которое её поднимет.
