#### §4.09 Упражнения на понятия

Пять упражнений на применение понятий Раздела 4. Цель — перейти от «я понял» к «я умею применять».

---

**Упражнение 1. Карта фаз**

Заполните таблицу для трёх доменов. Для каждого определите фазу по трём признакам:

| Домен | Какие вопросы задаю? | Что считаю очевидным? | Какие связи вижу? | Фаза |
|-------|----------------------|-----------------------|-------------------|------|
| Работа | | | | |
| Семья | | | | |
| Здоровье | | | | |

Выберите домен с самой низкой фазой. Какой один вопрос вы начнёте задавать, чтобы приблизиться к следующей фазе?

---

**Упражнение 2. S1/S2-дневник**

В течение одного дня отмечайте:

| Время | Ситуация | Какое мышление работало? (S1/S2) | Что помогло/мешало переключиться? |
|-------|----------|----------------------------------|-----------------------------------|
| | | | |

Найдите одну ситуацию, где S1 дал сбой, а S2 было нужно. Что можно изменить в среде, чтобы в следующий раз включилось S2?

---

**Упражнение 3. Системность vs систематичность**

Выберите одну регулярную практику (например, утренний ритуал, ежедневное чтение, спорт).

- Опишите практику: что, когда, как часто?
- Это систематичность (регулярность) или системность (связь с целью)?
- Если практика систематична, но не системна — как её соединить с общей картиной?
- Если практика системна, но не систематична — как сделать её регулярной?

---

**Упражнение 4. Объект или описание?**

Возьмите 5 вещей из своего рабочего стола, календаря или текущей задачи. Для каждой определите: материальный объект или описание?

| Вещь | Объект / Описание | Тест: останется ли, если убрать всех людей? |
|------|-------------------|---------------------------------------------|
| | | |
| | | |
| | | |
| | | |
| | | |

Особое внимание: задача в трекере, отчёт о работе, спецификация продукта, документ на диске, программа. Какие из них объекты, а какие — описания? Где у вас в работе путаница между уровнями?

---

**Упражнение 5. Заземление трёх абстрактных понятий**

Выберите 3 абстрактных понятия из последнего занятия или из своей работы (например: «культура команды», «эффективность», «качество», «системное мышление», «приоритет»).

Для каждого заполните три опоры:

| Понятие | Материальный объект (носитель) | Наблюдаемое действие | Проверяемый признак (за неделю) |
|---------|-------------------------------|---------------------|--------------------------------|
| | | | |
| | | | |
| | | | |

Где не получилось заполнить все три колонки — это понятие у вас на уровне 0-1 заземления (см. §4.05). Это нормально как старт; ненормально — оставить как есть. Что нужно сделать на следующей неделе, чтобы поднять понятие на уровень 2-3?
