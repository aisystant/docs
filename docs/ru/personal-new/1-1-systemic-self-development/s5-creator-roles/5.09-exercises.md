#### §5.09 Упражнения на понятия

Три упражнения на применение понятий Раздела 5. Цель — перевести «я понял пять ролей» в «я вижу их в своей неделе и могу с ними работать».

---

**Упражнение 1. Аудит ролей за прошедшую неделю**

Возьмите календарь и блокнот. Для каждого блока времени, занявшего минимум 30 минут, проставьте роль и продукт:

| День | Время | Что делал | Роль | Рабочий продукт (если был) |
|------|-------|-----------|------|------|
| | | | | |
| | | | | |
| | | | | |

После заполнения посчитайте: сколько часов было в роли Ученика, Интеллектуала, Профессионала? Какая роль не появилась ни разу? Если по итогам недели не появилось ни одного рабочего продукта — Профессионал был неактивен, даже если казалось, что вы много работали.

---

**Упражнение 2. Перевод неудовлетворённости в задачу**

Это упражнение на роль Интеллектуала. Возьмите одну текущую неудовлетворённость средней силы.

| Шаг | Что делать | Запись |
|-----|------------|--------|
| Исходная формулировка | Как ощущается «в сыром виде» | |
| О чём именно «не так» | Симптом → возможная причина | |
| Признак того, что стало лучше | Что я увижу, по чему пойму | |
| Кто что должен сделать | Действие и исполнитель | |
| К какому сроку | Конкретная дата | |
| Где сейчас застрял | Какой шаг блокирует движение | |

Сравните исходную формулировку и итоговую. Если итоговая ничем не лучше — Интеллектуал ещё не сработал, нужен второй заход или партнёр для сверки.

---

**Упражнение 3. Театральная метафора на чужой критике**

Это упражнение на роль Ученика и инструмент театральной метафоры. Вспомните одну недавнюю критику вашей работы (за последний месяц), которая задела вас лично — не только по делу.

- В какой роли вы получили эту критику? (Профессионал, Интеллектуал, Ученик — какая роль выпускала тот продукт)
- К чему относилась критика — к роли (исполнению, методу, качеству продукта) или к вам как личности?
- Если критика была к роли — какой именно метод или приём в исполнении провалился? что в нём подтянуть?
- Если критика была к личности — это уже не профессиональная обратная связь; как разговор стоило бы переформулировать?

Записанный разбор оставьте у себя. Через две недели вернитесь к нему и проверьте: появилось ли изменение в вашем поведении или в подходе к критике?
