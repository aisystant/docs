#### §3.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 3. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **IWE** | Среда интеллектуальной работы: интерфейс между биологическим мозгом и экзокортексом, объединяющий хранилище, процессинг и правила взаимодействия | §3.01 |
| **Интерфейс экзокортекса** | Слой, через который мозг взаимодействует с внешней когнитивной системой; снижает нагрузку на входе и повышает качество на выходе | §3.01 |
| **Pack** | Личная база знаний и принципов с адресацией, связями и возвратом; хранилище экзокортекса | §3.02 |
| **Личная база знаний** | Система хранения информации, созданная, структурированная и используемая самим человеком | §3.02 |
| **Агент** | Сущность в IWE, выполняющая определённую когнитивную операцию с материалом Pack | §3.03 |
| **Роль агента** | Чётко определённая функция агента внутри системы, граничащая его ответственность | §3.03 |
| **MCP** | Протокол передачи контекста между компонентами IWE; нервная система экзокортекса | §3.04 |
| **Контекстный протокол** | Соглашение о формате, содержании и правах передаваемых данных | §3.04 |
| **Протокол** | Повторяющаяся последовательность шагов с триггером, входом, результатом и правилом завершения | §3.05 |
| **ОРЗ** | Открытие — Работа — Закрытие: базовый протокол сессии взаимодействия с экзокортексом | §3.05 |
| **Цифровой двойник** | Структурированная модель человека в Pack: профиль, история, паттерны, фокус, ограничения | §3.06 |
| **RCS-профиль** | Модель компетенций и характеристик для калибровки траектории развития | §3.06 |
| **Аудит IWE** | Систематическая проверка трёх уровней среды с картой сильных и слабых мест | §3.07 |
