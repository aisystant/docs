#### §4.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 4. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Траектория развития** | Осознанный путь человека как Созидателя в техноэволюции | §4.01 |
| **Стратегия саморазвития** | Выбор фокуса на основе неудовлетворённостей, ценностей и калибра | §4.01 |
| **Роль** | Функциональная единица деятельности: набор действий, методов и ответственности | §4.02 |
| **Исполнитель роли** | Человек, который в данный момент выполняет действия роли | §4.02 |
| **Ролевое мастерство** | Степень владения методами роли; предсказуемость результата | §4.02 |
| **Потребитель** | Агент, использующий технологии без понимания и без участия в передаче | §4.03 |
| **Позиция в техноэволюции** | Функциональная роль относительно технокультурного процесса | §4.03 |
| **Неудовлетворённость** | Несоответствие между текущим состоянием и желаемым; сигнал для развития | §4.04 |
| **Топливо развития** | Энергия, которую даёт правильно интерпретированная неудовлетворённость | §4.04 |
| **Ценность** | Операционный критерий выбора; то, что защищается ресурсами в конфликте | §4.05 |
| **Фильтр выбора** | Способ отсеивать возможности, которые не ведут в выбранном направлении | §4.05 |
| **Калибр** | Масштаб задач, которые человек способен взять, удержать и довести до результата | §4.06 |
| **Масштаб задач** | Уровень сложности, ресурсоёмкости и продолжительности задачи | §4.06 |
| **Стратегическая сессия** | Регулярная практика калибровки траектории из четырёх блоков | §4.07 |
