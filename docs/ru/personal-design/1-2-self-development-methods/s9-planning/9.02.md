#### §9.02 Планирование и стратегирование: два крыла полёта

**Время:** 45 мин чтение + 25 мин

**В одном предложении:** Стратегирование отвечает на зачем и куда, планирование — на что и когда; без стратегии план — список дел, без плана стратегия — мечта.

**Мем, который снимается.** "Я много планирую — значит, у меня есть стратегия." Планирование без стратегии — бег по кругу. Стратегия без планирования — парение в облаках. Два крыла полёта должны работать вместе.

**Определение из источника.**
**Стратегирование — направление, планирование — движение.** Стратегия редко меняется. План адаптируется еженедельно.

**Планирование — метод, не инструмент.** Календарь и таск-менеджер — инструменты. Планирование — метод мышления о времени и приоритетах.

**Каскад связывает уровни.** Вектор ВДВ → месячный план → недельный план → дневные слоты. Каждый уровень детализирует предыдущий.

**Развитие мысли.**
Пилоты часто склоняются в одну из крайностей: либо много планируют без ясности направления (бег по кругу), либо много стратегируют без конкретных шагов (парение в облаках). Два крыла полёта должны работать вместе: стратегия задаёт направление, план — движение. Каскад соединяет эти уровни.

**Метод — минимальный шаг.**
1. Откройте текущий план недели. (~5 мин)
2. Для каждого пункта спросите: как это связано с вектором ВДВ? (~10 мин)
3. Если нет связи — либо уберите пункт, либо найдите связь. (~5 мин)
4. Зафиксируйте в Pack. (~5 мин)

Проверить текущий план недели: каждый пункт связан с вектором ВДВ? Если нет — либо убрать пункт, либо найти связь.

**Пример из жизни.** Инженер тридцати лет имел подробный план на неделю (15 пунктов), но ни один не был связан со стратегией. Проверка: «эта задача ведёт к вектору ВДВ?» — показала, что 9 из 15 пунктов — реакция, не движение. Он убрал 6 пунктов, добавил 2, связанных с вектором. Неделя стала продуктивнее при меньшем объёме.

**Степени мастерства:**
| Степень | Что происходит | Критерий перехода |
|---------|----------------|-------------------|
| 1. Объяснение | Только планирую или только стратегирую | — |
| 2. Умение | Понимаю различие, но не поддерживаю связь | Один раз практически применил |
| 3. Навык | Регулярно проверяю связь плана со стратегией | Регулярно применяю без подсказки |
| 4. Мастерство | Управляю каскадом стратегия-план через Pack с автоматической проверкой связности | Помогаю другим освоить, передаю культуру |

**Проверка себя.**
-  У меня есть и стратегия, и план, и они связаны
-  Я отличаю стратегический уровень от операционного
-  Мои ежедневные действия ведут в направлении векторов ВДВ

**На практике.**
Через IWE Pack разделяет стратегические документы (каскад ВДВ) и операционные (планы, WP-реестр), обеспечивая связь между ними.
