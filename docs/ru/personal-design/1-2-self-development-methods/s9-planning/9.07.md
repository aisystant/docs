# IWE как система планирования

## В одном предложении

IWE Pack обеспечивает единое пространство для стратегии, планирования и исполнения через шаблоны каскада, WP-реестр и агент-Диагност с отчётами по выполнению.

## Почему это важно

Планирование разрознено: стратегия в одном документе, план в календаре, задачи в таск-менеджере, отчёты в голове. IWE объединяет это в одну систему: каскад ВДВ связан с WP-реестром, план недели — с каскадом, факт — с планом. Агент-Диагnostic анализирует выполнение и предлагает корректировки.

## Различение

**Pack — операционная система, не просто хранилище.** Pack не только хранит документы, но и связывает их между собой: WP → вектор → намерение.

**WP-реестр — поток работ, не список желаний.** Каждый WP имеет формулировку, статус, срок и связь с каскадом. Без связи WP теряет смысл.

**Агент-Диагност — отчёт, не оценка.** Диагност показывает тренды и паттерны, не судит о «успешности».

## Связь с IWE

Pack содержит шаблоны каскада (год, квартал, месяц, неделя, день) со связями между уровнями. WP-реестр — все рабочие продукты с формулировкой, статусом, сроком, связью с каскадом. Агент-Диагност — отчёт по выполнению плана, анализ отклонений, тренды.

## Как применить

1. Откройте Pack и найдите WP-реестр.
2. Проверьте: все ли WP имеют формулировку, статус, срок и связь с каскадом?
3. Дополните недостающие поля для 3 WP.
4. Запустите отчёт Диагностика по план/факт.

## Практика (25 мин)

Проверить в Pack: есть ли WP-реестр? Все ли WP имеют формулировку, статус, срок и связь с каскадом? Дополнить недостающие поля для 3 WP.

## Самопроверка

- [ ] У меня есть WP-реестр в Pack
- [ ] Каждый WP связан с элементом каскада
- [ ] Я использую отчёты Диагностика для анализа выполнения

## Степени мастерства

| Степень | Что происходит | Критерий перехода |
|---------|----------------|-------------------|
| 1. Объяснение | Не использую Pack для планирования | — |
| 2. Умение | Храню планы в Pack, но не связываю с каскадом | Один раз практически применил |
| 3. Навык | Системно веду WP-реестр с связями и статусами | Регулярно применяю без подсказки |
| 4. Мастерство | Интегрирую планирование, исполнение и анализ в единую систему IWE | Помогаю другим освоить, передаю культуру |
