#### §10.04 Дрейф методов: почему методы размываются

**Время:** 45 мин чтение + 25 мин

**В одном предложении:** Без регулярной проверки методы естественно размываются: защищённый слот сдвигается, медленное чтение превращается в скоростное, досуг — в инерцию; дрейф — не провал, а явление, которому противостоит регулярный аудит.

**Мем, который снимается.** "Метод перестал работать — он плохой." Методы размываются без регулярной проверки. Степень "умение" требует практики, иначе метод опускается до "объяснения". Дрейф — естественное явление.

**Определение из источника.**
**Дрейф — отклонение, не провал.** Методы размываются у всех — это естественный процесс. Провал — не заметить дрейф.

**Профилактика — проверка, не парanoia.** Ежемесячный аудит методов — рутина, не признак проблемы.

**Восстановление быстрее освоения.** Вернуть метод на прежнюю степень мастерства легче, чем осваивать с нуля. Но только если дрейф замечен вовремя.

**Развитие мысли.**
Методы не сохраняются сами по себе. Степень мастерства «умение» требует регулярной практики, иначе метод опускается до «объяснения». Пилот думает, что «всё ещё умеет», но на деле метод размыт. Проверка раз в месяц — минимум для стабильности. Без неё система методов медленно деградирует, пока bottleneck не блокирует всё.

**Метод — минимальный шаг.**
1. Проверьте 3 метода, которые практиковались 3+ месяца назад. (~5 мин)
2. На какой степени они сейчас? (~10 мин)
3. Если опустились — напишите план восстановления. (~5 мин)
4. Зафиксируйте в Pack. (~5 мин)

Проверить 3 метода, которые практиковались 3+ месяца назад. На какой степени они сейчас? Если опустились — написать план восстановления. Зафиксировать в Pack.

**Пример из жизни.** "Метод перестал работать — он плохой." Пилот 6 месяцев назад освоил метод чтения (степень «умение»). Сейчас читает редко, заметок нет. Аудит показал: метод дрейфнул до «объяснения». План восстановления: 2 недели ежедневного чтения по 20 минут с фиксацией. Метод вернулся на степень «умение».

**Степени мастерства:**
| Степень | Что происходит | Критерий перехода |
|---------|----------------|-------------------|
| 1. Объяснение | Не замечаю дрейфа, удивляюсь, когда метод «перестал работать» | — |
| 2. Умение | Замечаю дрейф, но не системно | Один раз практически применил |
| 3. Навык | Провожу ежемесячный аудит и восстанавливаю методы | Регулярно применяю без подсказки |
| 4. Мастерство | Системно управляю стабильностью методов через Диагност и Pack | Помогаю другим освоить, передаю культуру |

**Проверка себя.**
-  Я проверяю степень мастерства по методам не реже раза в месяц
-  Могу назвать методы, которые сейчас дрейфуют
-  У меня есть план восстановления для дрейфующих методов

**На практике.**
Через IWE агент-Диагност проводит ежемесячный аудит методов: статус, тренды, дрейф, рекомендации по восстановлению.
