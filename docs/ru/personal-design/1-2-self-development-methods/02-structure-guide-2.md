### 1.01 Метод как форма: как, а не что

```yaml
subsection_id: PD.GUIDE.2.S1.SS1
parent_section_id: PD.GUIDE.2.S1
title: "Метод как форма: как, а не что"
cp_check: [cp.skl]
bh_check: [bh.met]
bottleneck_hint: "Если застрял - проверь bh.met: есть ли хотя бы один метод на уровне умение?"
```

**Понятия:** Метод (форма деятельности, отделенная от содержания) -> Форма -> структура действия -> способ. Форма vs содержание -> структура/материя -> различение.

**Содержание:** Метод - это форма (как я действую), а не содержание (что я думаю). Один метод работает в разных доменах, потому что форма универсальна, а содержание меняется. Метод первичен, IWE - усиление, не замена. Через IWE метод фиксируется в Pack и отслеживается как элемент культуры работы.

**Содержание:** Культура ученика - базовые практики, необходимые для применения методов: слот, саморегуляция, фиксация. Простые практики дают 90% эффекта, но сопротивление переходу длится месяцы - это эффект Земмельвейса. Минимальный шаг: один защищенный слот, одна неделя, одна проверка. Через IWE агент-Диагност напоминает о культурных практиках и фиксирует прогресс.

**Понятия:** Степени мастерства -> уровень развития -> фаза. Итерации и инкременты -> пошаговый рост -> накопление.

**Содержание:** Четыре ступени: Объяснение (могу рассказать) -> Умение (могу сделать под присмотром) -> Навык (могу сделать стабильно) -> Мастерство (могу адаптировать и учить). Диагностика текущей степени по каждому методу: шкала 0-3. Правило: пока метод не на уровне умение, не брать следующий. Через IWE Pack-шаблон Дашборд 9 методов визуализирует степени мастерства.

Как выбрать: анализ текущей ступени + bh-индексы + неудовлетворенности. Через IWE агент-Диагност ежемесячно рассчитывает bottleneck-метод по bh/cp.


**Понятия:** Микрорешения -> маленький шаг -> элемент цикла. Пайплайн микрорешений -> последовательность -> конвейер. Итерации и инкременты -> пошаговый рост -> накопление.

**Содержание:** Развитие идет не скачками, а через микрорешения - минимальные осознанные шаги, которые можно повторить. Пайплайн микрорешений: решение -> действие -> фиксация -> проверка -> следующее решение. Каждая итерация дает инкремент - маленький, но устойчивый рост. Через IWE Pack фиксирует каждое микрорешение как элемент WP-реестра.

**Практика (25 мин):** Сформулировать одно микрорешение для текущего bottleneck-метода. Оно должно занимать не более 15 минут и быть повторяемым в течение недели.

Простые практики (слот, заметка, pomodoro) дают 90% эффекта, но сопротивление длится месяцы. Быстрые удовольствия - антипаттерн, который конкурирует с методами за внимание. Без потребления нет входа, без размышлений нет переработки, без действия нет выхода. Через IWE Pack-шаблон Бюджет времени показывает план/факт по 5 классам.

### 3.01 Чтение для мышления, а не для информации

### 3.02 Образовательная заметка: строительный блок знания
