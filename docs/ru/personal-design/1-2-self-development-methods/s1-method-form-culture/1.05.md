#### §1.05 Фокус на узком месте: один метод, максимальный эффект

**Время:** 45 мин чтение + 25 мин = 70 мин

**В одном предложении:** человек одновременно «осваивает» десять методов, но доводит до навыка ни одного — и удивляется, почему не растёт; узкое место системы саморазвития — метод с наименьшей степенью мастерства, и его подъём на одну ступень даёт больше, чем запуск трёх новых.

**Сигнатура понятий:**

- **Задаётся:** Узкое место метода — метод саморазвития, который находится на самой низкой степени мастерства среди всех методов в работе; ограничивает всю систему, потому что без него другие методы не получают необходимой поддержки
- **Задаётся:** Правило одного фокуса — принцип: в один момент времени работать с одним узким методом, доводя его до следующей степени мастерства, прежде чем брать новый; аналог правила одного ограничения в теории ограничений
- **Используется:** Теория ограничений — метод управления тремя потоками созидателя; производительность системы определяется узким местом, а не суммарной мощностью всех элементов; формализована в Pack
- **Используется:** Степени мастерства — четырёхступенчатая шкала; узкое место — метод на степени объяснения или умения, который блокирует переход системы на следующий уровень

**Мем, который снимается.** «Чем больше методов я осваиваю одновременно, тем быстрее расту. Если читать, писать, планировать, считать время и стратегировать — хоть что-то да приживётся.» На самом деле каждый метод требует внимания как дефицитного ресурса. Десять методов на степени объяснения дают нулевой рабочий продукт. Один метод на степени навыка даёт стабильный результат и создаёт инфраструктуру для следующего. Распыление — не диверсификация, а фрагментация.

**Определение из источника.** Узкое место в системе саморазвития определяется по тем же правилам, что и узкое место в производственной системе: это элемент с наименьшей пропускной способностью, который ограничивает выход всей системы. Ключевое различие теории ограничений: оптимизировать можно только узкое место. Ускорение потоков, которые уже не ограничивают, не даёт прироста — система всё равно упирается в бутылочное горлышко. Применительно к методам: человек, который совершенствует планирование на степени навыка, вместо того чтобы поднять чтение со степени объяснения, тратит внимание на не-ограничивающий поток.

**Развитие мысли.** Инкремент Guide 2 по сравнению с Guide 1: если Guide 1 ввёл теорию ограничений как способ понимания системы в целом (три потока: рабочие продукты, информация, энергия), то Guide 2 §1.05 применяет эту теорию к конкретному методу саморазвития. Узкое место здесь — не поток в целом, а один метод внутри потока, который находится на самой низкой степени мастерства.

Связь со степенями мастерства прямая. Метод на степени объяснения — узкое место, потому что он не производит рабочий продукт. Метод на степени умения — ещё узкое место, потому что требует усилия и нерегулярен. Метод на степени навыка — уже не узкое место, он работает сам. Метод на степени мастерства — создаёт избыточность, которую можно направить на подъём другого метода.

Правило одного фокуса решает проблему распыления. Человек одновременно пробует читать систематически, вести заметки, использовать Pomodoro, планировать неделю и писать рефлексии. Диагностика по степеням мастерства показывает: чтение — объяснение, заметки — объяснение, Pomodoro — умение, планирование — объяснение, рефлексии — объяснение. Узкое место — не Pomodoro, который уже работает, а любой из методов на объяснении. Но выбрать нужно один, потому что подъём двух методов с объяснения на умение одновременно требует вдвое больше внимания — а внимание уже дефицитно.

**Метод — минимальный шаг.** Практика «Выбор узкого метода» (25 мин):
1. Перечислите методы саморазвития, которые вы сейчас «осваиваете» — не планируете освоить, а уже пробуете (5 мин).
2. Определите степень мастерства каждого метода по шкале: объяснение, умение, навык, мастерство (10 мин).
3. Найдите узкое место — метод с наименьшей степенью. Если таких несколько — выберите тот, который блокирует остальные (например, защищённый слот блокирует всё остальное) (5 мин).
4. Сформулируйте правило: «Две недели не беру новых методов, работаю только с [название метода], пока он не поднимется на следующую степень» (5 мин).

**Пример из жизни.** Специалист двадцати девяти лет одновременно внедряет учёт времени, систематическое чтение, ведение заметок и планирование недели. Через месяц ничего не работает стабильно: учёт времени ведётся три дня из семи, чтение сорвалось, заметки есть, но беспорядочные, планирование делается под конец недели. Диагностика по степеням: учёт времени — умение, чтение — объяснение, заметки — объяснение, планирование — объяснение. Узкое место — не учёт времени. Он решает: две недели фокус только на защищённом слоте для чтения. Критерий перехода на умение: три сессии чтения по 45 минут в неделю с фиксацией. Через две недели чтение на умении — у него есть материал для заметок. Заметки начинают расти сами, потому что появилось, что фиксировать. Планирование становится легче, потому что есть содержание недели.

**Типичная ошибка.** «Я застрял на методе — значит, он мне не подходит, нужно найти другой.» Люди склонны interpretировать низкую степень мастерства как несовместимость метода с их «типом личности», а не как нормальный этап прохождения. На самом деле застревание на объяснении — не признак неподходящего метода, а признак недостатка внимания, вложенного в этот метод. Другая ошибка: «Подниму сразу два метода, они же связаны.» Связь методов реальна, но подъём требует внимания. Два метода на подъёме конкурируют за один и тот же дефицитный ресурс — и оба застревают.

**Степени мастерства:**

| Степень | Что происходит | Критерий перехода |
|---------|----------------|-------------------|
| 1. Объяснение | Могу сформулировать, что такое узкое место метода, и отличить его от «метода, который мне не нравится». | — |
| 2. Умение | Нахожу узкое место среди своих методов и выбираю один для фокуса на две недели; не беру новых методов в этот период. | — |
| 3. Навык | Правило одного фокуса работает автоматически; при застревании первым делом проверяю степень мастерства, а не меняю метод. | — |
| 4. Мастерство | Помогаю другому человеку найти его узкое место и удержать фокус на одном методе, не давая распылиться. | — |
**Проверка себя.**
- Сколько методов я сейчас «осваиваю» одновременно — и сколько из них на степени объяснения?
- Если бы я мог выбрать только один метод на ближайшие две недели — какой дал бы максимальный эффект для всей системы?
- Какой метод блокирует остальные: без него другие методы не работают или работают хуже?

**На практике.** Возьмите лист бумаги. Напишите в столбик все методы саморазвития, которые вы сейчас пробуете. Против каждого поставьте: О, У, Н или М. Найдите все методы с О. Выберите один — тот, без которого остальные работают хуже. Подчеркните его. Напишите под ним: «Две недели только этот метод. Новые — после». Это и есть ваше узкое место.

**Что дальше.** Узкое место показывает, на чём фокусироваться. Но даже с правильным фокусом человек может пытаться делать слишком много за раз — прыгать от одного метода к другому без итераций.
