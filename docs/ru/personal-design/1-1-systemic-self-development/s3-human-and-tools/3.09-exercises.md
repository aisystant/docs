#### §3.09 Упражнения на понятия

Три упражнения на применение понятий Раздела 3. Цель – перейти от «я понял» к «я умею применять».

---

**Упражнение 1. Диагностика Болида**

Прочитайте три описания. Для каждого определите: что слабее – Пилот, Машина или контур обратной связи?

> *А. Игорь точно знает, чего хочет, и может объяснить это в три предложения. Но он работает в хаосе: заметки в пяти приложениях, задачи в голове, нет ни одного регулярного протокола.*
>
> *Б. Марина имеет отлично настроенные инструменты: система заметок, автоматизация, чёткие процессы. Но она не может объяснить, зачем всё это, и каждый месяц меняет направление.*
>
> *В. Дмитрий и цели ясны, и инструменты есть. Но он никогда не сверяет план с фактом: сколько часов реально работал, сколько задач завершил, какие сроки сорвал.*

Запишите: слабый компонент у каждого, и один шаг для улучшения.

---

**Упражнение 2. Четыре природы**

Оцените свою текущую интеллектуальную среду по четырём природам (шкала 0–3):

| Природа | Оценка | Что проявляет? | Что отсутствует? |
|---------|--------|----------------|------------------|
| Наставник | | | |
| Мастерская | | | |
| Со-творец | | | |
| Зеркало | | | |

Выберите природу с наименьшей оценкой. Какое одно действие задействует её на этой неделе?

---

**Упражнение 3. Метод без среды**

Выберите один метод, который вы считаете важным (например, ведение заметок, планирование дня, рефлексия).

- Опишите метод тремя шагами.
- Проверьте: работает ли он без вашего основного цифрового инструмента? Как?
- Если нет – метод не освоен. Опишите минимальную среду, в которой метод всё равно работает.
