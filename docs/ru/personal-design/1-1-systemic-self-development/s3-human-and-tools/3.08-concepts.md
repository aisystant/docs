#### §3.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 3. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Болид** | Система «Пилот + Машина», в которой человек и инструменты работают как единое целое | §3.01 |
| **Пилот** | Человек как управляющий центр: мышление, намерения, роли, ценности, выбор направления | §3.01 |
| **Машина** | Всё, что усиливает человека: инструменты, среды, привычки, протоколы | §3.01 |
| **Мировоззрение как машинный код** | Совокупность принципов, через которые человек интерпретирует реальность и принимает решения | §3.02 |
| **Мышление = моделирование** | Системное мышление — построение моделей объекта из понятий и приёмов, а не запоминание фактов | §3.02 |
| **Зависимость модели от строителя** | Один и тот же объект порождает разные модели у разных моделеров — у каждого свой набор понятий и приёмов | §3.02 |
| **Метод как runtime** | Алгоритм действий, воспроизводимый независимо от инструментов | §3.03 |
| **Среда исполнения** | Контекст, который усиливает метод, но не заменяет его | §3.03 |
| **Пять ролей Машины** | Пять способов усиления Пилота: рабочее место, костюм-расширитель, связь с сообществом, выращиваемый инструмент, проводник | §3.04 |
| **Контур инструментов** | Способ организации, при котором инструменты работают как система | §3.05 |
| **Обратная связь Болида** | Механизм, при котором Машина показывает Пилоту данные для корректировки курса | §3.06 |
| **Слепые зоны Пилота** | Области, в которых Пилот не видит реальность объективно | §3.06 |
| **Аудит Болида** | Систематическая проверка Пилота, Машины и расхождения между ними | §3.07 |
