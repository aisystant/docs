#### §10.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 10. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Калибр личности** | Шкала масштаба деятельности: от «Я» до «человечество и поколения»; определяет контур внимания и ответственности | §10.01 |
| **Траектория развития** | Четыре этапа: интеграция → дифференциация → трансценденция → слияние с культурой; путь расширения калибра | §10.01 |
| **Проект системного саморазвития** | Инженерный подход к личности как к целевой системе: диагностика, проектирование, реализация, проверка | §10.02 |
| **Саморазвитие ≠ курсы** | Саморазвитие — системная работа над собой как целевой системой; не набор курсов, а инженерный проект | §10.02 |
| **Элита** | Сообщество, ориентированное на SoTA; среда, в которой нормой является движение к лучшим знаниям и практикам | §10.03 |
| **Культ лучших знаний (SoTA)** | Норма сообщества: постоянное обновление знаний до state of the art; не поклонение авторитетам, а ориентация на лучшее | §10.03 |
| **SoTA (state of the art)** | Современный уровень области: лучшие подходы и открытые вопросы; точка отсчёта для любой деятельности | §10.03 |
| **Русло цивилизации** | Направление техноэволюции: куда движется человечество в долгосрочной перспективе; контекст для выбора траектории | §10.04 |
| **Антихрупкость цивилизации** | Способность цивилизации не просто выживать при потрясениях, а становиться сильнее; критерий устойчивости | §10.04 |
| **Удача** | Непредсказуемое событие, которое может быть как положительным, так и отрицательным; не контролируется, но может быть использована | §10.05 |
| **Подготовка почвы** | Состояние готовности: быть в форме, чтобы распознать удачу и использовать её; то, что контролируется | §10.05 |
| **Расслоение общества** | Интеллектуальное меньшинство создаёт системы, потребляющее большинство ими пользуется; тренд, который усиливается | §10.06 |
| **Создатели vs потребители** | Различение: создатели производят системы и контексты, потребители ими пользуются; не моральная оценка, а функциональное описание | §10.06 |
| **Проект траектории** | Синтез всех элементов Руководства 1: калибр, роли, ценности, методы, IWE в единый жизненный проект | §10.07 |
| **Жизненный проект** | Долгосрочная программа развития, объединяющая все домены жизни в единую траекторию | §10.07 |
