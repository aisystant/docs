#### §4.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 4. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Мировоззренческая дуга** | Эволюция мировоззрения через пять фаз от «непонятно» до «я в мире – деятель» | §4.01 |
| **Фаза мировоззрения** | Линза, через которую человек интерпретирует реальность в данный момент | §4.01 |
| **Спектр формальности мышления** | Шкала от животного рефлекса до машинной точности | §4.02 |
| **Материальный объект ≠ описание** | 4D-онтология: всё либо физ. объект, либо описание; третьего нет | §4.02 |
| **Вопрос-якорь «описание чего?»** | Дисциплинирующий приём: к любому понятию спрашивать, к какому материальному объекту оно относится | §4.02 |
| **Быстрое мышление** | Автоматический, интуитивный режим; требует мало энергии, склонен к ошибкам | §4.03 |
| **Медленное мышление** | Осознанный, аналитический режим; требует энергии, даёт точность | §4.03 |
| **Собранность** | Способность переключаться с быстрого на медленное мышление и удерживать его | §4.03 |
| **Систематичность** | Регулярность, дисциплина, повторяемость действий | §4.04 |
| **Системность** | Способность видеть систему: связи, причины, следствия, границы | §4.04 |
| **Системное мировосприятие** | Стратегия восприятия мира как системы с границами и связями | §4.05 |
| **Деятельностный кругозор** | Широта взглядов за пределы своего домена | §4.05 |
| **Заземление понятия** | Связывание абстрактного понятия с конкретным материальным объектом, действием и проверяемым признаком | §4.05 |
| **Уровни заземления (0-4)** | Шкала освоения понятия от цитирования до передачи через материальный пример | §4.05 |
| **Принцип оптимизма Дойча** | Всё, что не запрещено законами физики, возможно | §4.06 |
| **Диагностика фазы** | Практика определения текущей фазы мировоззрения по трём признакам | §4.07 |
