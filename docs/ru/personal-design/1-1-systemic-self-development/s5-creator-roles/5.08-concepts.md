#### §5.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 5. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Ролевая траектория созидателя** | Пирамида из пяти ролей; не последовательность стадий, а смещение фокуса инвестиций | §5.01 |
| **Роль Ученик** | Функция, в которой объект внимания — ты сам как система; единственная роль, направленная внутрь | §5.01 |
| **Метароль** | Роль, без которой остальные роли работают хаотично или превращаются в имитацию | §5.01 |
| **Роль ≠ должность** | Роль — функция в конкретный момент, а не строка в штатном расписании | §5.01 |
| **Объект роли** | То, на что направлено внимание в данной роли; у каждой из пяти ролей — свой объект, физический или ментальный | §5.01 |
| **Роль Интеллектуал** | Функция, в которой объект внимания — проблемы и неопределённости, ещё не ставшие задачами | §5.02 |
| **Проблема ≠ Задача** | Проблема = ступор, метод неизвестен; Задача = метод известен, исполнитель назначен, критерий определён | §5.02 |
| **Мышление как инструмент** | Отношение к мышлению как к набору приёмов, которые можно тренировать и заменять; качество мышления — характеристика Интеллектуала, а не его объект | §5.02 |
| **Перевод неопределённости в задачу** | Главная операция Интеллектуала: смутное ощущение → формулировка с критериями готовности | §5.02 |
| **Роль Профессионал** | Функция, в которой объект внимания — задача и рабочий продукт; предметная область — среда, а не объект | §5.03 |
| **Рабочий продукт** | Зафиксированный выход системы, который существует отдельно от исполнителя | §5.03 |
| **Деятельность ≠ продукт** | Деятельность измеряется временем, продукт — готовым артефактом, переданным дальше | §5.03 |
| **Роль Исследователь** | Функция, в которой объект внимания — фронтир: граница между тем, что в области уже известно, и тем, что ещё нет | §5.04 |
| **Фронтир (граница знания)** | Линия между общепринятым и неизвестным/спорным в области; движется со временем | §5.04 |
| **Современный уровень области** | Текущее состояние поля: лучшие подходы и открытые вопросы | §5.04 |
| **Мастерство ≠ информация** | Мастерство накапливается с практикой, информация обесценивается | §5.04 |
| **Роль Просветитель** | Функция, в которой объект внимания — идеи и смыслы для общества; задача — чтобы они воспроизводились независимо от автора | §5.05 |
| **Репликация системы** | Устройство передачи, при котором результат не зависит от присутствия автора | §5.05 |
| **Передача культуры** | Передача способа обращения с фактами, привычек, норм; медленнее передачи знаний | §5.05 |
| **Театральная метафора** | Взгляд на жизнь как на смену ролей с тренируемым качеством исполнения | §5.06 |
| **Роль отделена от исполнителя** | Критика и неудача относятся к ролевому исполнению, а не к личности целиком | §5.06 |
| **Психологическая гигиена роли** | Практика, при которой неудача в роли не разрушает самооценку | §5.06 |
| **Ритуал входа в роль** | Короткая последовательность: роль / метод / продукт / признак готовности | §5.06 |
| **Карта ролей** | Таблица «домены жизни × пять ролей» с оценкой каждой клетки от 0 до 3 | §5.07 |
| **Уровень охвата (калибр личности)** | Шкала от «Я» до «человечество и поколения», описывающая контур деятельности | §5.07 |
