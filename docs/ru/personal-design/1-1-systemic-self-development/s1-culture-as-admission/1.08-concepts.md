#### §1.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 1. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Культура ученика** | Инфраструктура из шести конкретных практик, которая удерживает методы обучения в работе независимо от мотивации | §1.01 |
| **Системное саморазвитие** | Подход к обучению как к непрерывному процессу со структурой, ритмом и инструментами – в отличие от эпизодических попыток | §1.01 |
| **Допуск к сложности** | Минимальный набор базовых практик, без которого сложные методы не усваиваются; аналог асептики для хирурга | §1.02 |
| **6 элементов культуры ученика** | Защищённый слот, саморегуляция, фиксация мышления, восстановление, гигиена информации, ритуал входа – конкретный список практик, образующих инфраструктуру | §1.02 |
| **Квадрант компетенций** | Четырёхстадийная модель освоения навыка: неосознанная некомпетентность → осознанная некомпетентность → осознанная компетентность → неосознанная компетентность | §1.03 |
| **Степень мастерства** | Четырёхступенчатая шкала глубины освоения: объяснение → умение → навык → мастерство | §1.03 |
| **Стиль жизни ученика** | Режим, в котором саморазвитие встроено в распорядок жизни как норма – не «проект на февраль», а постоянный ритм | §1.04 |
| **Слот саморазвития** | Фиксированный временной блок, специализированный по типу содержимого: предназначен именно для деятельности саморазвития (не для работы, отдыха или семьи) | §1.04 |
| **Отрицательная полезность труда** | Экономический принцип: встроенное предпочтение досуга перед трудом; объясняет прокрастинацию как рациональный сигнал о невыгодном соотношении выигрыш/затраты | §1.05 |
| **Защищённый слот** | Атрибут защищённости, применимый к слоту любого типа содержимого (саморазвития, стратегической сессии, восстановления и т.д.); три компонента – время, триггер, правило «не переносится, а пропускается» | §1.06 |
| **Саморегуляция** | Навык замечать собственное состояние до того, как оно упало ниже рабочего порога, и корректировать действие до потери эффективности | §1.06 |
| **Шкала состояния** | Инструмент самонаблюдения 0–5 для оценки рабочего состояния; используется для решения «что делать в слоте» при разном уровне энергии | §1.06 |
| **Аудит культуры ученика** | Диагностическая практика оценки шести элементов культуры по шкале 0–10; выявляет слабейший элемент и задаёт фокус на 4 недели | §1.07 |
