#### §1.09 Упражнения на понятия

Три упражнения на применение понятий Раздела 1. Цель — перейти от «я понял» к «я умею применять».

---

**Упражнение 1. Диагностика стадии**

Прочитайте три описания поведения. Для каждого определите: на какой стадии квадранта компетенций находится человек?

> *А. Антон начал вести заметки. Ему тяжело — он часто забывает, пишет нерегулярно. Но он видит, что без записей теряет важные мысли.*
>
> *Б. Мария уже два года ведёт дневник размышлений. Она не задумывается, когда записывать — это происходит само. Коллеги удивляются, как она всё успевает отслеживать.*
>
> *В. Денис слышал про ведение заметок на курсе. Он считает, что ему это не нужно: «у меня хорошая память».*

Запишите: какая стадия у каждого? Почему?

---

**Упражнение 2. Уравнение прокрастинации**

Вы откладываете одно конкретное дело уже несколько дней. Пройдите через уравнение отрицательной полезности труда:

1. Что именно откладываете? Запишите одним предложением.
2. Каков ожидаемый выигрыш? (Оцените: насколько ощутим выигрыш прямо сейчас, не через месяц?)
3. Каковы затраты на вход? (Время, усилие, дискомфорт — насколько они ощущаются как большие?)
4. Что можно сделать, чтобы увеличить ожидаемый выигрыш?
5. Что можно сделать, чтобы уменьшить затраты на вход?
6. Какое минимальное действие вы можете сделать прямо сейчас?

---

**Упражнение 3. Карта культуры ученика**

Вспомните трёх человек из вашего окружения: одного, кто регулярно чему-то учится; одного, кто начинает и бросает; и себя.

Для каждого из трёх попробуйте оценить состояние шести элементов культуры (0–3 вместо 0–10 для краткости):

| Элемент | Человек 1 (регулярный) | Человек 2 (начинает и бросает) | Я |
|---------|----------------------|-------------------------------|---|
| Защищённый слот | | | |
| Саморегуляция | | | |
| Фиксация мышления | | | |
| Восстановление | | | |
| Гигиена информации | | | |
| Ритуал входа | | | |

Какой паттерн вы видите? Что отличает первого от второго? Что это говорит о вашем собственном профиле?
