#### §9.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 9. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Сбалансированность личности** | Состояние, в котором 4 характеристики (агентность, мастерство, ресурсность, эмоциональная стабильность) развиты пропорционально | §9.01 |
| **Агентность** | Способность инициировать и доводить до результата; характеристика личности, измеряемая через наблюдаемые действия | §9.01 |
| **Мастерство** | Накопленная способность качественно выполнять сложные задачи; растёт через практику, а не через информацию | §9.01 |
| **Ресурсность** | Способность восстанавливаться и поддерживать энергию; зависит от гигиенического минимума | §9.01 |
| **Эмоциональная стабильность** | Способность сохранять работоспособность при стрессе и неопределённости; не подавление, а управление | §9.01 |
| **6 результатов программы ЛР** | Ясность, агентность, собранность, регулярность, способность производить, продуктивность; критерии оценки прогресса | §9.02 |
| **Гигиенический минимум** | Шесть столпов инфраструктуры личности: сон, питание, движение, связи, досуг, смысловая работа | §9.03 |
| **Три потока личной жизни** | Познавательный (интеллект), переживательный (эмоции), созидательный (действие); структура жизненного пространства | §9.04 |
| **Засор потока** | Ситуация, когда один из трёх потоков блокирован или перегружен; bottleneck баланса | §9.04 |
| **Киберхарактеристики** | Шесть характеристик цифровой среды: агентность, интеллектуальная развитость, стрессоустойчивость, ресурсность, техноинтеграция, калибр | §9.05 |
| **Стиль жизни ученика** | Операционная система человека: совокупность привычек, ритуалов и структур, которые определяют, как протекает день | §9.06 |
| **Тройной контур** | Структура: био-ядро (тело), экзокортекс (инструменты), ИИ (внешний интеллект); три уровня поддержки деятельности | §9.06 |
| **Диагностика баланса** | Комплексная оценка: 4 хар-ки + 6 результатов + 6 столпов + 3 потока; даёт полную картину состояния системы | §9.07 |
| **Дашборд личности** | Визуальное представление диагностики: числа и тренды по всем измерениям | §9.07 |
