#### §6.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 6. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Ступень развития** | Режим функционирования человека как системы; диагностируется по четырём признакам (слот, методы, продукты, обратная связь), а не по самооценке | §6.01 |
| **Мастерство** | Способность производить результат вне зависимости от состояния; растёт через практику, а не через потребление информации | §6.01 |
| **5 ступеней развития** | Случайный → Практикующий → Систематический → Дисциплинированный → Проактивный; каждая ступень — режим функционирования, не грейд или награда | §6.01 |
| **Ступень ≠ грейд** | Ступень описывает режим функционирования человека, а не его социальный статус или должность | §6.01 |
| **Ворота развития** | Точка перехода между ступенями; открываются только при двойной проверке | §6.02 |
| **Двойная проверка** | Переход подтверждается с двух сторон: что вы делали (поведение) и что вы понимаете; одной стороны недостаточно | §6.02 |
| **Окно ступени** | Характерное время прохождения ступени в неделях; если окно закрылось, а переход не состоялся — сигнал узкого места | §6.03 |
| **Застревание на ступени** | Ситуация, когда окно ступени закрылось, но переход не произошёл; индикатор неподнятого узкого места | §6.03 |
| **Узкое место** | Один признак ступени (слот, метод, продукт или обратная связь), который блокирует переход и определяет скорость всего развития | §6.04 |
| **Теория ограничений (Goldratt)** | Метод управления, в котором улучшение всей системы начинается с устранения узкого места, а не с оптимизации всех процессов | §6.04 |
| **Профориентация** | Практическое исследование интересов и способностей как основа выбора ролей, а не «подходящей профессии» | §6.05 |
| **Выбор ролей** | Сбор ролей из неудовлетворённостей: какая роль закроет конкретную неудовлетворённость, с проверкой через эксперимент | §6.05 |
| **Степени мастерства** | Шкала освоения метода: Объясняю → Умею → Навык → Мастерство; каждая степень имеет критерий перехода | §1.03 |
| **Квадрант компетенций** | Глубина освоения метода: неосознанная некомпетентность → осознанная некомпетентность → осознанная компетентность → неосознанная компетентность | §6.06 |
| **План перехода** | Дорожная карта: цель-ступень, узкое место, действия на 4 недели, дата двойной проверки | §6.07 |
| **Точка проверки** | Дата, на которую назначена двойная проверка прохождения ворот развития | §6.07 |
