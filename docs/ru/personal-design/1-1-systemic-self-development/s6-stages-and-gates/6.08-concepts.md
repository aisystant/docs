#### §6.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 6. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Мастерство (главный капитал vs информация)** | Главный капитал развития — не информация, а способность применять знание; мастерство растёт через практику, а не через потребление контента | §6.01 |
| **5 ступеней развития** | Случайный → Пробующий → Систематический → Проактивный → Мастер; каждая ступень — режим функционирования, не грейд или награда | §6.01 |
| **Ступень ≠ грейд** | Ступень описывает режим функционирования человека, а не его социальный статус или должность | §6.01 |
| **bh-порог** | Минимум поведенческих навыков, необходимый для перехода на следующую ступень; проверяется Аттестатором | §6.02 |
| **cp-порог** | Минимум понятийного пространства, необходимый для перехода; проверяется Диагностом | §6.02 |
| **Двойной gate** | Процедура валидации перехода: Аттестатор подтверждает bh, Диагност подтверждает cp; оба должны дать согласие | §6.02 |
| **Окно ступени** | Характерное время прохождения ступени в неделях; если окно закрылось, а переход не состоялся — сигнал bottleneck | §6.03 |
| **Застревание на ступени** | Ситуация, когда окно ступени закрылось, но переход не произошёл; индикатор неподнятого bottleneck | §6.03 |
| **Bottleneck (узкое место)** | Один bh или cp, который блокирует переход на следующую ступень; определяет скорость всего потока развития | §6.04 |
| **Теория ограничений (Goldratt)** | Метод управления, в котором улучшение всей системы начинается с устранения узкого места, а не с оптимизации всех процессов | §6.04 |
| **Профориентация** | Диагностика интересов и способностей как основа для выбора ролей, а не «подходящей профессии» | §6.05 |
| **Выбор ролей** | Стратегия распределения внимания между пятью ролями созидателя на основе неудовлетворённостей и намерений | §6.05 |
| **Степени мастерства** | Шкала освоения: Объясняю → Умею → Навык → Мастерство; каждая степень имеет критерий перехода | §6.06 |
| **Квадрант компетенций** | Модель обучения: неосознанная некомпетентность → осознанная некомпетентность → осознанная компетентность → неосознанная компетентность | §6.06 |
| **План перехода** | Дорожная карта: что поднять (bh + cp), за какое время, как проверить; содержит дату gate | §6.07 |
| **Gate (точка проверки)** | Контрольная точка, в которой проверяется готовность к переходу на следующую ступень | §6.07 |
