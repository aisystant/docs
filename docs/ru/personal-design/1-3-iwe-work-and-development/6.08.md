#### §6.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 6. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Мультисессионность** | Управляемое чередование 2–4 фокусных РП с полным carry-over контекста между сессиями | §6.01 |
| **Многозадачность** | Хаотичное переключение между задачами без закрытия контуров и без управления контекстом | §6.01 |
| **Carry-over** | Перенос незавершённого РП на следующую сессию с полным контекстом, промежуточным результатом и точкой возобновления | §6.01, §6.02 |
| **Незакрытый контур** | РП, оставленный без фиксации промежуточного результата, создающий когнитивный долг | §6.02 |
| **Промежуточный результат** | Чёткая фиксация состояния РП на момент остановки: что сделано, открытые вопросы, следующий шаг | §6.02 |
| **Мультипликатор** | Отношение суммы бюджетов закрытых РП к физическому времени; целевой диапазон 1.5–2.5x | §6.03 |
| **Бюджет РП** | Запланированная трудоёмкость рабочего продукта, измеряемая во времени | §6.03 |
| **Физическое время** | Длительность работы, измеряемая часами | §6.03 |
| **Фокусный РП** | Рабочий продукт, которому выделено фокусное время с чётким бюджетом и критерием закрытия | §6.04 |
| **Оптимум** | Точка баланса между недостаточным параллелизмом и избыточными переключениями; индивидуален | §6.04 |
| **Стоп-лист** | Набор критериев прерывания РП, зафиксированных заранее, до начала работы | §6.05 |
| **Бюджет исчерпан** | Достижение запланированного времени на РП без закрытия; сигнал к переносу | §6.05 |
| **Состояние ниже порога** | Снижение bh.stb до уровня, при котором качество решений критически падает | §6.05 |
| **Coordination cost** | Сумма накладных расходов мультисессионности: переключение, синхронизация, управление очередностью | §6.06 |
| **Параллелизм** | Одновременное продвижение нескольких РП за счёт разнесения активного и пассивного времени | §6.06 |
| **Контекстное переключение** | Когнитивный процесс смены фокуса с одного РП на другой | §6.06 |
| **Anti-pattern** | Типичная ошибка, выглядящая разумной в краткосрочной перспективе, но ведущая к системному ухудшению | §6.07 |
| **Когнитивная авария** | Состояние, при котором число открытых контуров превышает когнитивную ёмкость пилота | §6.07 |
| **Жёсткое ОРЗ** | Дисциплина работы, при которой каждый РП закрывается или переносится целиком | §6.07 |
