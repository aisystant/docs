# Личное развитие (v4)

> **Статус:** в разработке. Новые универсальные руководства программы ЛР в формате v4 (reference).
>
> **Старые курсы:** см. [`/ru/personal/`](/ru/personal/) — сохранены для текущих участников.

## Состав программы v4

1. **«Системное саморазвитие»** — мировоззрение, роли, траектория, ценности (ступени 1–5)
2. **«Методы саморазвития»** — M001–M008: учёт времени, чтение, мышление, досуг, окружение, стратегирование, планирование (ступени 2–5)
3. **«IWE: работа и развитие»** — ОРЗ, Pack, агенты, MCP, протоколы (ступени 2–5)
4. **«Введение в системное мышление (ВСМ)»** — различения, роли, границы, эмерджентность, системный разбор (ступени 3–5)

## Отличие от v3 (старые курсы)

| | v3 (старые курсы) | v4 (новые руководства) |
|---|---|---|
| **Формат** | Курс с недельным темпом | Справочник (reference) + структура для Портного. Домашки и задания остаются — они адаптируются Портным под пилота |
| **Навигация** | Линейная: неделя 1 → неделя 10 | Нелинейная: искать по нужде, но с рекомендуемым порядком от Портного |
| **Персонализация** | Общий для всех | Проекция через Портного под RCS-профиль |
| **IWE** | **Не было** — ни инфраструктуры, ни агентов, ни протоколов | Отдельное руководство 3 с 14 элементами культуры работы |
| **ВСМ** | Встроено в 1-1 как §6 | Отдельное руководство 4 |
