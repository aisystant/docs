#### §1.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 1. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Системное мышление** | Мышление с понятиями трансдисциплин интеллект-стека: восприятие → модель → представление | §1.01 |
| **Систематичность** | Регулярность, дисциплина, порядок в действиях: поведение → паттерн → структура | §1.01 |
| **Системность** | Способность «разрезать» мир на системы: восприятие → анализ → декомпозиция | §1.01 |
| **Различение** | Операция, фиксирующая границу между понятиями; атом мышления | §1.02 |
| **Граница понятий** | Линия раздела между двумя понятиями, проведённая мыслью, а не данная природой | §1.02 |
| **Интеллект-стек** | Шестнадцать трансдисциплин, прошивающих интеллект: знание → архитектура → фундамент | §1.03 |
| **Трансдисциплина** | Дисциплина, прошивающая все предметные области: метод → универсальность → инвариант | §1.03 |
| **Инструмент мышления** | Метод или модель, извлечённый из трансдисциплины и применимый к любому объекту для получения нового вида знания | §1.03 |
| **Мировоззрение** | Инвариантные различения, через которые интерпретируется мир: программа → рамка → фильтр | §1.04 |
| **Машинный код личности** | Глубинные убеждения, управляющие восприятием: конфиг → интерпретатор → поведение | §1.04 |
| **Проблема** | Неопределённость, неструктурированная ситуация: неопределённость → разрыв → напряжение | §1.05 |
| **Задача** | Определённость, структурированная ситуация с критериями приёмки: определённость → форма → действие | §1.05 |
| **Системный разбор** | Метод перевода проблемы в задачу: метод → трансформация → инструмент | §1.05 |
| **Работа** | Вид деятельности, определяемый ролью и методом: деятельность → дисциплина → поток | §1.06 |
| **Управление работами** | Координация потока работ, а не список задач: управление → поток → система | §1.06 |
| **Метод** | Система деятельности, состоящая из дисциплины, инструментов и практик: дисциплина → система → способ | §1.07 |
| **Дисциплина** | Совокупность знаний о том, как действовать в определённой области: знание → теория → основа | §1.07 |
| **Инструмент** | Артефакт, усиливающий способность агента: артефакт → посредник → расширение | §1.07 |
