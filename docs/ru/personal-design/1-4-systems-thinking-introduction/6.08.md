#### §6.08 Список понятий раздела 6

Сводная таблица всех понятий, введённых в Разделе 6. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Целевая система** | Система, для которой создаётся решение; то, что должно работать лучше благодаря проекту; отвечает на вопрос «для чего мы это делаем?» | §6.01 |
| **Получатель** | Роль, для которой функция целевой системы имеет ценность; находится вне целевой системы | §6.01 |
| **Надсистема** | Система более высокого уровня, в которую входит целевая система; создаёт ограничения и возможности | §6.02 |
| **Среда надсистемы** | Условия, которые надсистема создаёт для целевой системы: ресурсы, ограничения, критерии приёмки | §6.02 |
| **Влияние надсистемы** | Действие надсистемы на целевую: разрешённое, обязательное и запрещённое | §6.02 |
| **Система создания** | Система, которая создаёт целевую систему; включает роли, инструменты, процессы и материалы | §6.03 |
| **Цепочка создания** | Полная последовательность этапов от возникновения идеи до эксплуатации целевой системы | §6.03 |
| **Передача** | Момент, когда результат одного этапа становится входом следующего; критическая точка разрыва | §6.05 |
| **Ближнее окружение** | Системы, непосредственно взаимодействующие с целевой: поставщики, конкуренты, партнёры, пользователи | §6.04 |
| **Дальнее окружение** | Системы, влияющие на целевую опосредованно: регуляторы, тренды, культура, технологические сдвиги | §6.04 |
| **Экосистема** | Совокупность всех систем (целевой, надсистемы, окружения), взаимодействующих вокруг общей функции | §6.04 |
| **Система «я»** | Человек, рассматриваемый как система с подсистемами; каждый уровень имеет свою функцию и границу | §6.06 |
| **Подсистема «я»** | Функционально обособленный уровень системы «я»: тело, эмоции, мышление, социальные роли | §6.06 |
| **Расширенное «я»** | Система «я» плюс инструменты, отношения и цифровые идентичности, которые выполняют функции подсистем | §6.06 |
| **Мета-система** | Система, которая наблюдает, описывает, анализирует или управляет другой системой | §6.07 |
| **Мета-системный переход** | Переход внимания от работы внутри системы к работе на уровне мета-системы | §6.07 |
| **Уровень анализа** | Позиция наблюдателя относительно системы: внутри, на границе, на уровень выше | §6.07 |
