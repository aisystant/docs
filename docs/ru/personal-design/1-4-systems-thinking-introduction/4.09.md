#### §4.09 Упражнения на понятия

Три практических упражнения для закрепления понятий раздела 4.

---

**Упражнение 1. Тройное различение для одной системы (20 мин)**

Выберите систему, с которой вы работаете прямо сейчас (продукт, процесс, инструмент, команду).

Заполните таблицу для этой системы:

| Уровень | Вопрос | Ваш ответ |
|---------|--------|-----------|
| Воплощение | Что физически существует прямо сейчас? | |
| Описание | Как вы понимаете эту систему? | |
| Документация | В каких документах зафиксировано описание? Когда обновлялись? | |

После заполнения: найдите расхождения между уровнями. Что не совпадает?

---

**Упражнение 2. Ролевая перспектива (25 мин)**

Возьмите систему, которую знаете хорошо (продукт, процесс, команду).

1. Назовите три роли, которые взаимодействуют с этой системой.
2. Для каждой роли запишите: какие три вопроса эта роль хочет задать о системе? Что ей критически важно знать?
3. Найдите существующую документацию этой системы. Для каких ролей она написана? На чьи вопросы отвечает?

Вывод: напишите, для какой роли отсутствует ролевое описание.

---

**Упражнение 3. Карта интерфейсов (15 мин)**

Нарисуйте систему как прямоугольник. Не рисуйте внутренности.

1. Запишите 3–5 элементов окружения (другие системы, роли, ресурсы) вокруг прямоугольника.
2. Для каждой связи «система — окружение» нарисуйте стрелки и подпишите: что передаётся и через какой конкретный модуль.
3. Найдите пробелы: есть ли важное для системы окружение, с которым нет модуля взаимодействия?
