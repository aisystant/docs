#### §6.09 Упражнения на понятия

Три практики на применение понятий раздела 6. Каждая — на конкретный проект или ситуацию из вашей жизни.

---

**Практика 1: Карта системного анализа (15 мин)**

Выберите один текущий проект или задачу. Заполните таблицу:

| Элемент | Ваш ответ |
|---------|-----------|
| Целевая система (для чего мы это делаем?) | |
| Получатель (кто пользуется результатом?) | |
| Надсистема (в чём это живёт?) | |
| Три ограничения от надсистемы | 1. 2. 3. |
| Система создания (кто и как это делает?) | |
| Три системы ближнего окружения | 1. 2. 3. |
| Три системы дальнего окружения | 1. 2. 3. |

После заполнения спросите: какой элемент был сложнее всего определить? Это — ваше слепое пятно в системном анализе.

---

**Практика 2: Цепочка создания и точки разрыва (15 мин)**

Возьмите завершённый проект (или проект, который «застрял»). Восстановите цепочку создания:

| Этап | Что произошло | Кто участвовал | Результат передан? |
|------|---------------|----------------|-------------------|
| Идея | | | |
| Концепция | | | |
| Проектирование | | | |
| Производство | | | |
| Эксплуатация | | | |

Отметьте этапы, где передача не состоялась или состоялась с потерями. Сформулируйте: что именно потерялось (информация, качество, намерение)? Какой интерфейс передачи нужен, чтобы этого не повторилось?

---

**Практика 3: Мета-вопрос к текущей проблеме (10 мин)**

Выберите проблему, которую вы пытаетесь решить более часа без прогресса.

1. Запишите вопрос, который вы сейчас задаёте себе. (2 мин)
2. Переформулируйте его на уровень выше: не «как решить X?», а «почему мой способ решения X не работает?» или «какая система порождает X?» (3 мин)
3. Ответьте на мета-вопрос. Зафиксируйте инсайт. (3 мин)
4. Сформулируйте одно действие на уровень выше. (2 мин)

Зафиксируйте результат в Pack: исходная проблема, мета-вопрос, инсайт, действие.
