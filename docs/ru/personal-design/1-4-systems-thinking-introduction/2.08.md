#### §2.08 Список понятий раздела

Сводная таблица всех понятий, введённых в Разделе 2. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Системный подход 1.0** | Классическая парадигма: система — объект физического мира, выделенный наблюдателем, но существующий независимо от него | §2.01 |
| **Целостность** | Система обладает свойствами, не присущими её элементам по отдельности; целое ≠ сумма частей | §2.01 |
| **Вложенность** | Иерархическая организация систем: каждая система состоит из подсистем и входит в надсистемы | §2.01 |
| **Эмерджентность** | Появление новых свойств целого, не присущих частям; предсказуема через моделирование, но не сводима к анализу частей | §2.01 |
| **Системный подход 2.0** | Субъектно-интересная парадигма: система существует только при наличии заинтересованного лица, которое проводит границу по своему интересу | §2.02 |
| **Ролевой интерес** | Интерес роли, не личный, определяющий релевантные свойства системы для данной роли | §2.02 |
| **Граница системы** | Линия раздела между системой и окружением; в СП 2.0 проведена по интересу, а не по физическому периметру | §2.02 |
| **Заинтересованное лицо** | Роль с интересом к объекту, от которого зависят границы и свойства системы | §2.02 |
| **Системный подход 3.0** | Безмасштабная и деантропоморфная парадигма: закономерности работают на любом уровне, системы не «хотят» и не «стремятся» | §2.03 |
| **Безмасштабность** | Закономерности системного подхода работают от атома до галактики, от индивида до цивилизации | §2.03 |
| **Деантропоморфность** | Отказ от антропоцентризма в описании систем; системы функционируют, а не «действуют» | §2.03 |
| **Мета-системный переход** | Переход на уровень надсистемы, когда проблема не решается в рамках текущей системы | §2.03 |
| **Создатель видения** | Роль, формулирующая описание системы до её физического существования | §2.04 |
| **Предпринимательская гипотеза** | Проверяемое утверждение о целевой системе: для кого она, какие свойства должна иметь, как будет измеряться успех | §2.04 |
| **Предприниматель** | Роль видения: кто видит возможности и формулирует образ будущего | §2.04 |
| **Целевая система** | Система, для которой создаётся решение; объект проектирования | §2.04 |
| **Расширенное предприятие** | Система, границы которой проведены по цепочке создания, а не по юридическому периметру | §2.05 |
| **Цепочка создания** | Pipeline от потребности до эксплуатации; последовательность ролей, создающих ценность | §2.05 |
| **Система создания** | Система, создающая целевую систему; включает всех участников цепочки создания | §2.05 |
| **Надсистема** | Система, содержащая целевую систему; определяет контекст и ограничения | §2.05 |
| **Неустроенности** | Сигналы диссонанса между фактическим и нормативным состоянием; эмоциональная форма неудовлетворённостей | §2.06 |
| **Движущая сила** | Энергия, необходимая для преодоления инерции системы; без неё изменения невозможны | §2.06 |
| **Коллективное системное мышление** | Способность группы переходить от индивидуальных ролей к совместному моделированию системы, в которой эти роли взаимодействуют | §2.06 |
| **Жизненный цикл 1.0** | Классический водопад: линейные фазы с контрольными точками, последовательное прохождение стадий | §2.07 |
| **Жизненный цикл 2.0** | Эволюционная спираль: система развивается через циклы адаптации с обратной связью | §2.07 |
| **Совершенствование** | Улучшение в рамках существующей структуры; оптимизация без смены парадигмы | §2.07 |
| **Развитие** | Смена структуры системы; выход за рамки существующей парадигмы | §2.07 |
