#### §7.08 Список понятий раздела 7

Сводная таблица всех понятий, введённых в Разделе 7. Используйте её для проверки: можете ли вы объяснить каждое определение своими словами, не заглядывая в текст?

| Понятие | Краткое определение | Введено |
|---------|---------------------|---------|
| **Системный разбор** | Метод анализа системы через четыре последовательных вопроса: зачем, что, как, когда | §7.01 |
| **4 вопроса** | Полный цикл анализа: зачем (функция и интересы), что (границы и состав), как (процессы и ограничения), когда (временная структура) | §7.01 |
| **Последовательность** | Принципиальный порядок вопросов: зачем → что → как → когда; нарушение порядка разрушает осмысленность ответов | §7.01 |
| **Системная карта** | Визуальное представление системы с элементами, связями и границами; инструмент системного разбора | §7.02 |
| **Таблица 3×3** | Матрица анализа: три роли (создатель, оператор, пользователь) × три типа систем (целевая система, система создания, окружение) = 9 ячеек с предметами интереса | §7.03 |
| **Процесс** | Последовательность действий, преобразующих вход в выход; повторяется, не имеет чёткого начала и конца | §7.04 |
| **Система** | Совокупность элементов и связей, объединённых общей функцией; существует независимо от конкретного проекта | §7.04 |
| **Проект** | Временное усилие по созданию уникального результата; имеет начало, конец и чёткие критерии приёмки | §7.04 |
| **Стадия** | Крупный период проекта с чётким результатом (альфой) и критериями перехода | §7.05 |
| **Этап** | Часть стадии с конкретным рабочим продуктом и чеклистом приёмки | §7.05 |
| **Логическое время** | Последовательность стадий независимо от календаря; отвечает на вопрос «что должно быть готово до чего?» | §7.05 |
| **Физическое время** | Календарные даты и длительности; отвечает на вопрос «когда это будет сделано?» | §7.05 |
| **Альфа** | Ключевая сущность проекта, которая должна достичь определённого состояния для перехода к следующей стадии | §7.06 |
| **Рабочий продукт** | Конкретный артефакт, создаваемый на этапе, с чёткими критериями приёмки | §7.06 |
| **Приёмка** | Проверка рабочего продукта по чеклисту: соответствует ли он критериям цели и реализации | §7.07 |
| **Проверка цели** | Вопрос: «Решает ли продукт ту задачу, ради которой он создавался?» | §7.07 |
| **Проверка реализации** | Вопрос: «Соответствует ли продукт заявленным требованиям и стандартам качества?» | §7.07 |
