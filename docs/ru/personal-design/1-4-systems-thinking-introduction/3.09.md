#### §3.09 Упражнения на понятия

Три упражнения на применение понятий Раздела 3. Цель — перейти от «я понял» к «я умею применять».

---

**Упражнение 1. Ролевой анализ: роль, должность, исполнитель**

Выберите организацию или проект, в котором вы участвуете. Возьмите одного человека и разложите его на три компонента.

> *Пример: технический директор стартапа одновременно выполняет роль архитектора (проектирует систему), роль наставника (вводит новых участников в технологический стек) и роль переговорщика (согласует технические требования с клиентами). Его должность — «технический директор». Он один — исполнитель, но ролей три.*

Для выбранного человека:
1. Напишите его должность (официальное название позиции).
2. Перечислите все роли, которые он реально выполняет (не по описанию должности, а по факту деятельности).
3. Найдите: есть ли роли, которые противоречат друг другу? Есть ли роли, на которые не хватает времени?

Зафиксируйте результат. Как изменилось ваше понимание ситуации после разделения ролей и должности?

**Упражнение 2. Три границы для одной системы**

Выберите объект из вашей работы — проект, команду или процесс. Сформулируйте три разных вопроса об этом объекте, каждый из которых требует разной границы системы.

| Вопрос | Что входит в систему | Что остаётся в окружении |
|--------|---------------------|--------------------------|
| Вопрос 1: ... | | |
| Вопрос 2: ... | | |
| Вопрос 3: ... | | |

Сравните три границы: какая граница самая широкая? Какая даёт наибольшую управляемость? Какая открывает решения, невидимые при других границах?

---

**Упражнение 3. Функция, состав и переименование**

Возьмите три системы из вашей работы, которые сейчас называются нефункционально (например, «Отдел X», «Процесс Y», «Инструмент Z»). Для каждой:

1. Запишите текущее имя.
2. Сформулируйте функцию системы одним предложением: «Эта система существует для того, чтобы [функция]».
3. Предложите новое имя, которое отражает функцию: «[Что делает] [для кого/для чего]».
4. Спросите себя: является ли эта система функциональным или физическим объектом? Есть ли у неё единое физическое тело?

|-------------|---------|-----------|------------------------|
| | | | |
| | | | |
| | | | |

Зафиксируйте три переименования в Pack. Изменилось ли ваше понимание этих систем?
