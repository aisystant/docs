#### §7.09 Упражнения на понятия

Три практики на применение понятий раздела 7. Каждая — на конкретный проект или ситуацию из вашей жизни.

---

**Практика 1: Системный разбор по 4 вопросам (15 мин)**

Выберите одну текущую задачу или проект. Ответьте на 4 вопроса в строгой последовательности:

| Вопрос | Ваш ответ |
|--------|-----------|
| Зачем? (функция, интересы, получатель) | |
| Что? (границы, состав, элементы) | |
| Как? (процессы, ограничения, ресурсы) | |
| Когда? (стадии, этапы, критерии перехода) | |

После заполнения проверьте: нарушали ли вы порядок? Если ответили на «как» до «зачем» — перепишите. Порядок — не формальность, а защита от решения не той задачи.

---

**Практика 2: Таблица 3×3 для одной ситуации (15 мин)**

Выберите одну ситуацию из вашей текущей работы. Заполните таблицу 3×3:

| | Целевая система | Система создания | Окружение |
|---|---|---|---|
| **Создатель** | | | |
| **Оператор** | | | |
| **Пользователь** | | | |

После заполнения спросите: какие ячейки оказались сложнее всего? Это — области, где ваше понимание системы наиболее размыто.

---

**Практика 3: Приёмка одного рабочего продукта (10 мин)**

Возьмите последний рабочий продукт, который вы создали (документ, код, презентацию, отчёт).

1. Проверка цели: «Решает ли этот продукт ту задачу, ради которой он создавался?» Запишите ответ. (3 мин)
2. Проверка реализации: «Соответствует ли продукт заявленным требованиям?» Запишите ответ. (3 мин)
3. Если оба ответа «да» — продукт принят. Если хотя бы один «нет» — запишите, что нужно изменить. (4 мин)

Зафиксируйте результат в Pack: продукт, результат приёмки, доработки.
