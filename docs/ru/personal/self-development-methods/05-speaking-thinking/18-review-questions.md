# Вопросы для повторения и самопроверки

Ответьте письменно на вопросы. Используйте данные ответы для написания заготовки и опубликуйте её в клубе МИМ.

- Зачем деятелю мышление проговариванием?
- Чем мышление проговариванием отличается от риторики, мозгового штурма или дружеского общения?
- Какой рабочий продукт получается по итогам мышления проговариванием?
- На что необходимо обращать внимание в мышлении проговариванием?
- Перечислите основные принципы практики мышления проговариванием.
- Почему невозможно достичь мастерства в мышлении проговариванием без изучения трансдисциплин интеллект-стека?
- Мои заметки по изученному разделу.
