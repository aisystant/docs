# Вопросы для повторения и самопроверки

Ответьте письменно на вопросы. Используйте данные ответы для написания заготовки и опубликуйте её в клубе МИМ.

- Чем отличается работа от досуга?
- Что такое досуг?
- Чем отличается практика организации досуга от практик досуга?
- Перечислите принципы практики организации досуга и приведите примеры из личной жизни.
- С какими практиками саморазвития связана практика организации досуга?
- Почему важны практики сна и перерыва между помидорками?
- Мои заметки по изученному разделу.
