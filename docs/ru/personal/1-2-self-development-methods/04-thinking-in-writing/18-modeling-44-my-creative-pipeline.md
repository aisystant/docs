# 4.18. Моделирование 4.4. Мой творческий конвейер


Опишите свое текущее состояние мастерства по работе на творческом конвейере в части практики мышления письмом, а также укажите технологии, которыми вы пользуетесь для этого.

| **Стадии** | **Периодичность\*** | **Текущее состояние** | **Мой экзокортекс** |
| --- | --- | --- | --- |
| 1\. **Составление заметок:** мимолетные и/или образовательные | **Мимолетные заметки записываются в течение дня** в удобную программу на смартфоне. **Образовательные заметки составляются** в процессе систематического медленного чтения (желательно **каждодневного**). |  |  |
| 2\. **Разбор заметок:** разносите все заметки в черновики будущих текстов. | По возможности, но **не реже 1 раза в неделю** разбирать все заметки и разносить их в **черновики**. |  |  |
| 3\. **Выбор приоритетных тем черновиков для проработки:** темы выбираются исходя из работы и личных интересов в рамках приоритетных проектов. | **Во время личного стратегирования выбираются темы** из черновиков для проработки в течение определенного времени (недели, месяца). |  |  |
| 4\. **Написание заготовок из черновиков, и их публикация для единомышленников:** заготовка для своих, а публичный пост — для неограниченного круга лиц. | *Ежедневно работайте с приоритетными черновиками.** По мере возможности, но желательно **не реже 1 раза в неделю** в личном канале, в рубрике посты студентов в клубе МИМ, внутри рабочей команды или сообщества единомышленников. |  |  |
| 5\. **Используйте соответствующие практики оформления публичных материалов:** тексты для широкой аудитории. | Публичные тексты с использованием прикладных практик оформления текстов (инфостиль и т.п.) желательно публиковать **не реже 1 раза в месяц.** | . |  |
| 6\. **Продвигайте публичные материалы**: посты, модели и другие материалы, которые возникли как результат мышления письмом и практик оформления текстов. | Рабочие материалы необходимо распространять не только среди коллектива, но и среди профессионального сообщества. Кроме того, в целях повышения своего публичного статуса необходимо заниматься просветительской деятельностью. |  |  |
| 7\. **Не бойтесь возвращаться к ранее опубликованным заготовкам и текстам:** переписывайте их, создавая новые материалы. | **По необходимости.** |  |  |
