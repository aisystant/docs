# 6.5. Моделирование 6.1. Список информационных ресурсов. Метод организация досуга


Составьте список информационных ресурсов (каналов), которые сейчас находятся в вашем окружении или которые вы намерены включить в свое окружение, чтобы постоянно пропитываться, мотивироваться и получать знания по методу организация досуга. Необязательно иметь множество каналов, вполне достаточно если это будет руководство и клуб, но может у вас есть и другие. Например, каналы в телеграм по каким-то методам досуга.
