# 6.18. Вопросы для повторения и самопроверки


Ответьте письменно на вопросы. Используйте данные ответы для написания заготовки и опубликуйте её в клубе МИМ.

1. Чем отличается работа от досуга?
2. Что такое досуг?
3. Чем отличается практика организации досуга от практик досуга?
4. Перечислите принципы практики организации досуга и приведите примеры из личной жизни.
5. С какими практиками саморазвития связана практика организации досуга?
6. Почему важны практики сна и перерыва между помидорками?
7. Мои заметки по изученному разделу.
