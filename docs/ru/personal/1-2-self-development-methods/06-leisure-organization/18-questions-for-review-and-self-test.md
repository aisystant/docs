# 6.18. Вопросы для повторения и самопроверки


Ответьте письменно на вопросы. Используйте данные ответы для написания заготовки и опубликуйте её в клубе МИМ.

1. Чем отличается работа от досуга?
2. Что такое досуг?
3. Чем отличается метод организации досуга от методов досуга?
4. Перечислите принципы метода организации досуга и приведите примеры из личной жизни.
5. С какими методами саморазвития связана метод организации досуга?
6. Почему важны методы сна и перерыва между помидорками?
7. Мои заметки по изученному разделу.
