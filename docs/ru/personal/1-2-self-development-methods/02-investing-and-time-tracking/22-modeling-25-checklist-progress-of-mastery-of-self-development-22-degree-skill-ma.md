# Моделирование 2.5. Чек-лист "Прогресс мастерства саморазвития" 2.2: степень «Умение», мастерство инвестирования и учета времени


Данный чек-лист предназначен для тех, кто освоил практику инвестирования и учета времени на степень «Объяснения» и пытается наработать систематичность в применении этой практики. Он проверяет умение инвестировать и учитывать время на протяжении, как минимум, одного месяца. Вы можете использовать этот чек-лист по мере изучения следующих практик саморазвития и обращаться к нему время от времени.

В данном чек-листе достаточно написать «Да/Нет», но также можно ответить развернуто и даже дать ссылки на свои публикации в клубе МИМ, где вы демонстрируете знания по данному вопросу.

Для сведения: первая цифра «2» означает степень мастерства «Умение», вторая цифра «2» относится к разделу руководства, третья цифра — это порядковый номер основного вопроса, а четвертая цифра — это порядковый номер подвопроса для основного вопроса. На основной вопрос отвечаете «Да», если на все подвопросы ответ «Да».

| **2.2.1.** Использую **ритуал входа в роль и фокусировку в течение помидорки**. Могу сказать, что не делаю ничего важного без трекера Помодоро в течение, как минимум, двух недель (на этот вопрос можно ответить — «Да», если на все следующие подпункты 2.2.1.1-2.2.1.4 вы ответили «Да»). |  |
| 2.2.1.1. Протестировал в течение нескольких дней трекер и нашел свое оптимальное время фокусировки для учебного проекта и для рабочих проектов. |  |
| 2.2.1.2. Учусь по помидоркам в течение недели на обучение, как минимум, по 25 минут без отвлечений, соблюдаю ритуал и перерывы между ними для отдыха. |  |
| 2.2.1.3. Учитываю в течение недели время, как минимум, по двум проектам — Обучение и Работа (не выделяя других проектов и задач): соблюдаю ритуал, делаю перерыв между помидорками и нашел подходящее время помидорки. Однако, иногда бывают срывы. |  |
| 2.2.1.4. Исполняю в течение 2 недель ритуал по более чем двум проектам без срывов и забываний включения трекера. Всегда помню об итоговых рабочих продуктах, которые планирую получить по итогам завершения помидорки, всегда соблюдаю перерывы. |  |
| **2.2.2.** Веду в течение четырех недель **анализ инвестированного и учтенного времени,** **которое является мотиватором** в длительных проектах (на этот вопрос можно ответить — «Да», если на все следующие подпункты вы ответили «Да»). |  |
| 2.2.2.1. Учитываю не менее часа в день и за неделю получается не менее 7 часов общего учтенного времени по учебе. |  |
| 2.2.2.2. Учитываю не менее двух часов в день и за неделю получается не менее 12 часов учтенного времени по учебе и работе. |  |
| 2.2.2.3. Учитываю время не менее шести дней в неделю, а за неделю получается не менее 10 инвестиционных часов на учебу и более 30 часов общего учтенного времени. |  |
| 2.2.2.4. Выполняю в течение двух недель цель времени фокусировки на день 4 часов с вероятностью не менее 80%. |  |
| 2.2.2.5. Выполняю в течение двух недель цель времени фокусировки на день 5 часов с вероятностью более 80%. |  |
| **2.2.3.** Работаю три недели со всеми своими **приоритетными проектами и задачами,** а также умею настраивать под себя трекер Помодоро (на этот вопрос можно ответить — «Да», если на все следующие подпункты вы ответили «Да»). |  |
| 2.2.3.1. Настроил трекер Помодоро: цель времени фокусировки на день; при необходимости могу использовать разную длительность помидорки на разные задачи; другие количественные показатели практики. |  |
| 2.2.3.2. Могу работать с разной длительностью помидорки для разных задач до 45 минут. |  |
| 2.2.3.3. Могу работать с несколькими проектами и задачами в каждом проекте: не менее 3 проектов и не менее 2 задач в каждом. |  |
| **2.2.4.** Веду отдельно **список моих затруднений** по практике инвестирования и учета времени, и прорабатываю их раз в неделю, ставя гипотезы на неделю для тестирования. |  |
| **2.2.5.** Не говорю "потратил время" во всех случаях, а стараюсь осознанно выделять вложенное в будущее мастерство время и говорю в этих случаях **"инвестировал время".** |  |
| **2.2.6.** Использую практику инвестирования и учета времени **одновременно с другими практиками** саморазвития, как минимум, не занимаюсь без Помодоро практиками систематическое медленное чтение, мышлением письмом, организацией досуга, стратегирование, планирование. |  |
| **2.2.7.** Не менее 10 раз получилось реализовать **наградную помидорку,** расширяя границы возможного по количеству инвестированного и учтенного времени за день. |  |
