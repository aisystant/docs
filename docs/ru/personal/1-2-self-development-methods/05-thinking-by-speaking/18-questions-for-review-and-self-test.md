# 5.18. Вопросы для повторения и самопроверки


Ответьте письменно на вопросы. Используйте данные ответы для написания заготовки и опубликуйте её в клубе МИМ.

1. Зачем деятелю мышление проговариванием?
2. Чем мышление проговариванием отличается от риторики, мозгового штурма или дружеского общения?
3. Какой рабочий продукт получается по итогам мышления проговариванием?
4. На что необходимо обращать внимание в мышлении проговариванием?
5. Перечислите основные принципы практики мышления проговариванием.
6. Почему невозможно достичь мастерства в мышлении проговариванием без изучения трансдисциплин интеллект-стека?
7. Мои заметки по изученному разделу.
