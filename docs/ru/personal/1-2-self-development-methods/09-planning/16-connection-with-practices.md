# Как практика планирования связана с другими практиками саморазвития

Практика планирования тесно связана с другими практиками саморазвития и является их логическим продолжением.

**Связь со стратегированием:**
По итогам стратегирования определяются приоритетные проекты и общее время для получения какого-то результата. В планировании определяются формулировки работ, которые нужно вписать в график (в трекер Помодоро) и найти на них соответствующие ресурсы. В планировании используются данные из таблицы «Анализ и выбор приоритетных проектов» для составления бюджета времени.

**Связь с инвестированием и учетом времени:**
В процессе выполнения проекта используется практика учета и инвестирования времени, согласно которой важное внимание для личной мотивации уделяется выполнению плану по выделенному времени в рамках бюджета времени. Учет времени позволяет более обоснованно рассчитывать время на планируемые работы.

**Связь с мышлением письмом:**
Формулировки работ, планы, распорядок дня — все это создается мышлением письмом. Кроме того, в процессе планирования возникают мимолетные заметки с идеями по задачам и работам.

**Связь с систематическим медленным чтением:**
В плане работ обязательно должно быть время на обучение — изучение фундаментальных или прикладных курсов с помощью систематического медленного чтения.

**Связь с организацией досуга:**
При составлении распорядка дня и бюджета времени необходимо учитывать время на досуг и отдых между помидорками.

**Связь с формированием окружения:**
При запуске нового приоритетного проекта необходимо продумать, какое окружение потребуется для его реализации. Это становится отдельной задачей в плане работ.
