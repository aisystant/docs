# Рабочие продукты по практике планирования

На разных стадиях планирования появляются следующие типы рабочих продуктов:

- **Перечень задач** — список задач по приоритетным и текущим проектам, который ведется в моделере (например, Coda) и обновляется на сессиях стратегирования.

- **План работ** — список конкретных формулировок работ на определенный горизонт планирования (неделю и/или день), который ведется в трекере Помодоро.

- **Календарь со слотами времени** — инструмент для визуализации распределения работ по времени. Позволяет увидеть занятость и защитить важные временные промежутки.

- **Бюджет времени** — расчет общего времени на все плановые работы на неделю с разбивкой по типам (Важные/Текущие, Постоянные/Временные).

- **Распорядок дня** — описание режима жизни и привычек с указанием наиболее защищенных слотов времени для важных работ.

- **Список для крошек времени** — перечень небольших задач, которые можно выполнить в короткие промежутки свободного времени (5-15 минут).

- **Чек-листы** — списки мелких работ, которые собираются под общую формулировку работы. Позволяют не перегружать план работ мелкими деталями.

Все эти рабочие продукты связаны между собой и обновляются на сессиях стратегирования и планирования. По итогам применения практик стратегирования и планирования мы получаем план постоянных, временных, важных и текущих работ с бюджетом времени, а также при необходимости срочные работы.
