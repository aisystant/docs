# Моделирование 9.4. Мой распорядок дня

Для того, чтобы эффективнее распоряжаться временным ресурсом, желательно составить распорядок дня, который описывает режим жизни и привычки человека. Например, вы можете разбить день на несколько частей, определив наиболее защищенные слоты времени для того, чтобы поставить на них Важные работы.

При составлении распорядка дня необходимо обратить внимание на время отхода ко сну и на занятия, которые «убивают время». Кроме того, важно использовать так называемое правило «золотого часа» сразу после пробуждения.

Составьте свой распорядок дня, указывая временные промежутки и виды деятельности:

| Время | Вид деятельности | Важная/Текущая работа | Примечания |
|-------|-----------------|----------------------|------------|
| 6:00-7:00 | Пример: Подъем, утренние ритуалы | | |
| 7:00-8:00 | Пример: Завтрак, дорога на работу | | |
| 8:00-12:00 | Пример: Интеллектуальная работа, инвестиционные проекты | Важная | Защищенный слот времени |
| 12:00-13:00 | Пример: Обед, досуг | | |
| 13:00-18:00 | Пример: Операционная деятельность, текущие задачи | Текущая | |
| 18:00-20:00 | Пример: Досуг, семья | | |
| 20:00-22:00 | Пример: Подготовка ко сну | | |
| | | | |
