# 10.4. Моделирование 10.1. Чек-лист «Временной период по применению практик саморазвития»


Данный чек-лист поможет вам проверить, используете ли вы все практики саморазвития в разные периоды времени. Просто отметьте «Да» или «Нет» для каждой практики.

| **Временной период** | **Практика саморазвития** | **Задачи** | **Да/Нет** |
| --- | --- | --- | --- |
| Час | Инвестирование и учет времени | Включать трекер Помодоро, проводя ритуал и фокусируясь на работе (учебе). |  |
| Час | Организация досуга | Применять практики досуга, в том числе отдых между помидорками. |  |
| Час | Инвестирование и учет времени | Обязательно заканчивать фокусировку после окончания помодорки, чтобы отдохнуть. |  |
| День | Инвестирование и учет времени | Практиковаться в инвестировании и учете времени без выходных (начиная со степени «Навык»). |  |
| День | Систематическое медленное чтения | Изучать фундаментальные или прикладные курсы в зависимости от своей траектории развития и составлять образовательные заметки. |  |
| День | Мышление письмом | Записывать свои мысли в мимолетные и образовательные заметки. |  |
| День | Мышление письмом | Мыслить письмом в рамках своих рабочих и личных проектов, активно работая с черновиками, связанными с приоритетными задачами. |  |
| День | Организация досуга | Применять практики досуга на день |  |
| День | Планирование | Корректировать план работ и бюджет времени на день исходя из плана работ на неделю. |  |
| День | Инвестирование и учет времени | Вести учет количества учтенного и инвестированного времени в сравнении с целью фокусировки на день. |  |
| Неделя | Систематическое медленное чтения | На протяжении недели читать материалы для реализации приоритетных проектов с помощью медленного чтения и формировать образовательные заметки. |  |
| Неделя | Систематическое медленное чтения | Изучать новое знание по темам, которые фиксируются в списках дел по крошкам времени. |  |
| Неделя | Мышление письмом | Разобрать исчезающие заметки, формируя новые черновики или обогащая ранее созданные идеи новыми мыслями. |  |
| Неделя | Мышление письмом | Написать заготовки по рабочим и личным проектам для публикации в рабочем пространстве или в открытой сети для единомышленников (например, в личном блоге клуба МИМ или в телеграм-канале). |  |
| Неделя | Мышление письмом | Писать публичный пост, используя ИИ и дополнительные практики (блогера, ритора, просветителя и т.п.). |  |
| Неделя | Мышление письмом | Развивать свой творческий конвейер, в том числе оттачивать мастерство, развивать экзокортекс и т.п. |  |
| Неделя | Мышление проговариванием | Проговаривать понятия и принципы изучаемых теорий. |  |
| Неделя | Организация досуга | При необходимости определить практики досуга на неделю. Например, как проводить выходные. |  |
| Неделя | Организация досуга | Применять практики досуга на неделю |  |
| Неделя | Формирование окружения | Выполнять работы по формированию окружения |  |
| Неделя | Стратегирование | Провести сессию стратегирования и планирования для определения приоритетных проектов и задач, в том числе по развитию творческого конвейера, практик досуга, формирования окружения, списки дел для крошек времени. |  |
| Неделя | Стратегирование | Выполнять работы по адаптации под себя Порядка стратегирования и планирования |  |
| Неделя | Планирование | Выполнять работы по адаптации под себя Порядка стратегирования и планирования |  |
| Неделя | Планирование | Составить план работ и бюджет времени на неделю в рамках сессии стратегирования |  |
| Неделя | Инвестирование и учет времени | Вести учет количества учтенного и инвестированного времени в сравнении с целью фокусировки на неделю. |  |
| Неделя | Планирование | Публиковать заготовки и публичные посты |  |
| Месяц | Организация досуга | Применять практики досуга на месяц. |  |
| Месяц | Организация досуга | При необходимости определить практики досуга на месяц. Например, поездки или поход в театр. |  |
| Месяц | Формирование окружения | Определять приоритеты по изменению и/или развитию своего окружения |  |
| Месяц | Инвестирование и учет времени | Вести учет количества учтенного и инвестированного времени в сравнении с целью фокусировки на месяц. |  |
| Квартал | Организация досуга | Применять практики досуга на квартал. |  |
| Квартал | Организация досуга | При необходимости определить практики досуга на квартал. Например, сменить лыжи на велосипед или плавание. |  |
| Квартал | Стратегирование | Актуализировать свою траекторию развития, в том числе определяя курсы для обучения |  |
| Год | Инвестирование и учет времени | Вести учет количества учтенного и инвестированного времени в сравнении с целью фокусировки на год. |  |
| Год | Организация досуга | Применять практики досуга на год. |  |
| Год | Организация досуга | При необходимости определить практики досуга на год. Например, дальние путешествия. |  |
