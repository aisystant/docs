# Концептуальное пространство и выбор названий

В проектной команде обычно для каждого используемого слова можно указать плотность концептуального пространства: насколько все участники команды понимают это слово одинаково. Для простых физических объектов концептуальное пространство плотное, для сложных абстрактных — может быть очень неплотным.
