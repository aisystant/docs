# Референция и заземление: сначала осмысленность, потом истинность

К высказыванию нельзя сразу применять «правда/ложь», пока не понятно, к чему оно вообще отсылает.

Порядок проверки:

1. **Осмысленность**: у высказывания есть работающая референция? (C.2.1)
2. **Контекст**: в каком U.BoundedContext оно должно интерпретироваться? (A.1.1, E.10.D1)
3. **Заземление**: какие объекты и изменения в физическом мире должны наблюдаться? (C.2.1 + A.6.P)
4. **Истинность/адекватность**: совпадает ли сказанное с наблюдаемым?

Пример: фраза «Мы почти готовы к релизу».
