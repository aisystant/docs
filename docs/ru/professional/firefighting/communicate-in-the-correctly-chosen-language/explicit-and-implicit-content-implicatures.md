# Явное и неявное содержание. Импликатуры

В каждой реплике есть:

* **явное содержание** — то, что буквально сказано;
* **неявное содержание/речевая импликатура** — то, что достраивается из контекста, то, что имелось в виду.

Пример: «Мы, наверное, не уложимся к пятнице».

* Явно: есть риск по сроку.
* Неявно: срок уже точно сорван, можно перестать пытаться.
