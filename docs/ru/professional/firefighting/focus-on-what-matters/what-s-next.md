# Что дальше?

Вы закончили резидентуру по «Распожаризации». Рекомендуем попрактиковаться в применении принципов организации и выполнения работ, а затем пройти резидентуру «Моделирование как основа коммуникации и лидерства» для того, чтобы научиться:

* Моделировать рабочие процессы,
* Удерживать внимание команды на работе по моделям,
* Составлять такие объяснения, которые убедят команду сотрудничать с вами в ходе оргизменений.
