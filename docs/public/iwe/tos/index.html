Условия использования

Версия 1.0 · 11 мая 2026 г.

1. Общие положения

Настоящие Условия регулируют отношения между пользователем и ИП Церенов Цырен Иванович (далее — «Оператор») при использовании сервиса IWE (Intellectual Work Environment) и связанных продуктов: Telegram-бот @aist_me_bot, Knowledge Gateway, MCP-серверы.

Используя сервис, пользователь подтверждает согласие с настоящими Условиями. При несогласии — прекратить использование.

Условия могут изменяться. О существенных изменениях Оператор уведомляет за 14 дней через бот.

2. Описание сервиса

IWE — программная среда для интеллектуального труда, включающая:

  • Персональное руководство по саморазвитию
  • Цифровой двойник (агрегированный профиль)
  • Интеграции с базами знаний (GitHub, Google Calendar, WakaTime)
  • Доступ к курсам и контенту через подписку

Сервис предоставляется «как есть» (as-is) без гарантий бесперебойной работы.

3. Регистрация и идентификация

Регистрация — через Telegram или OAuth-провайдеров (Ory, Google, GitHub). Пользователь обязан предоставлять достоверные данные. Один пользователь — один аккаунт.

4. Подписка и платежи

Доступ к платным функциям — через подписку. Платежи через ЮKassa (РФ) или Stripe (международные, в разработке). Оператор не хранит данные банковских карт.

Возврат средств — по запросу в течение 14 дней с момента списания (Закон РФ «О защите прав потребителей», ст. 26.1). Запрос — через бот /support или email.

Подписка автопродлевается. Отмена — через бот /subscription.

5. Допустимое использование

Запрещено:

  • Использовать сервис для незаконных целей
  • Пытаться получить доступ к данным других пользователей
  • Реверс-инжиниринг, попытки обхода систем безопасности
  • Использовать AI-ответы для введения третьих лиц в заблуждение
  • Спам, флуд (>20 сообщений/мин активирует rate limiting)

Оператор вправе заблокировать аккаунт при нарушении без возврата средств.

6. Интеллектуальная собственность

Контент сервиса (методики, курсы, программы развития) — собственность Оператора.

Контент пользователя (документы, заметки, репозитории) — собственность пользователя. Оператор использует их только для предоставления сервиса.

7. Ограничение ответственности

Оператор не несёт ответственности за решения, принятые на основе AI-ответов, утрату данных при сбоях третьих сторон, косвенные убытки.

Совокупная ответственность Оператора ограничена суммой подписки за последние 6 месяцев.

AI-ответы сервиса — не медицинские, юридические или финансовые консультации.

8. Прекращение

Пользователь может прекратить использование через /unsubscribe + /mydata-delete. Оператор может прекратить сервис с уведомлением за 30 дней и возможностью экспорта данных.

9. Применимое право

Договор регулируется правом Российской Федерации. Досудебный порядок разрешения споров — обязательный (претензия по email с ответом в 30 дней).
