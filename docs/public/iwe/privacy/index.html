3. Согласие на обработку через AI

Claude Anthropic: сообщения обрабатываются через Claude API. Anthropic работает с нами по DPA и не использует данные для обучения моделей.

OpenAI Embeddings: при загрузке личных документов в базу знаний IWE — только с явного согласия при первом использовании. Без согласия функция недоступна.

8. Безопасность

TLS in-transit (HTTPS), AES-256 at-rest (Neon), Row-Level Security в БД, JWT-верификация на каждом запросе, rate limiting, ежедневный аудит безопасности.

9. Несовершеннолетние

Сервис не предназначен для лиц младше 16 лет. При обнаружении — напишите нам, мы удалим данные.

10. Изменения политики

При существенных изменениях — уведомление в боте за 14 дней.
