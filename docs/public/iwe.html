✦ Intellectual Work Environment

AI для каждого,
у кого есть дело

ИИ отлично справляется как эксперт по универсальным вопросам. Но постоянным коллегой, который помнит твой контекст, учится вместе с тобой и растёт в твоих проектах — это другой продукт и другой подход.

IWE — не ещё один чат с ИИ. Помнит твои проекты, замыслы и опыт. Формализует знания по структуре первых принципов. Подбирает мировой опыт под твой контекст.

Что такое IWE

Экзоскелет для ума, а не просто чатбот

IWE делает четыре вещи одновременно:

  • Помнит твои проекты, замыслы и опыт
  • Формализует знания по структуре первых принципов
  • Подбирает мировой опыт под твой контекст
  • Управляет ансамблем ИИ-агентов и людей под задачу

Со временем IWE будет знать твои дела в определённых деталях даже лучше тебя, но поможет удерживать всё целое вместе.

Инструкции подключения

Начните за 5 минут

Выберите удобный способ входа.

Telegram-бот — самый простой старт

Удобно для заметок в любой момент и старта погружения в системное саморазвитие прямо в мессенджере. Не требует сложных установок. Рекомендуется также для получения платформенных сообщений и бесед с ИИ, обученным системному мышлению.

Дальше — спроси бота дословно: «? Что нужно подключить для полного IWE?» (знак вопроса вначале). Для полного IWE потребуется подписка: оформи командой /buy.
  1. 1

    Найди бота в Telegram

    Открой Telegram и найди @aist_me_bot или перейди по ссылке.

  2. 2

    Нажми /start

    Бот возьмёт имя из Telegram и покажет меню. Аккаунт создаётся автоматически.

  3. 3

    Привяжи аккаунт Aisystant

    Введи команду /link — бот найдёт твой аккаунт автоматически или пришлёт ссылку для входа.

  4. 4

    Запусти Марафон

    Введи /learn — бесплатный старт без подписки.

Браузер — полная среда без установки

Удобно вне рабочего места — на телефоне, в дороге, на чужом компьютере. Стратегические сессии и общение с ИИ фиксируются в твоей базе знаний, включая личные проекты. Освоение программ МИМ через персональные руководства, которые можно сразу применять в работе.

Дальше — спроси Claude: «Как мне развиваться дальше?» или «Создай для меня персональное руководство по личному развитию, включая IWE». Claude запустит программу «Личное развитие» через твоё персональное руководство в Projects.
  1. 1

    Войди на Claude.ai

    Войди или зарегистрируйся на claude.ai.

  2. 2

    Открой настройки коннекторов

    Перейди в Настройки → Connectors и нажми «Add custom connector».

  3. 3

    Введи параметры Aisystant

    Name: Aisystant
    Remote MCP server URL: https://mcp.aisystant.com/mcp

  4. 4

    Подключись

    Нажми Add, затем Connect и пройди верификацию. Готово.

VS Code + Claude Code — среда для серьёзной работы

Полная среда у тебя на компьютере — накопление знаний, формализация опыта, персональные руководства МИМ, автономные агенты. Лучший выбор для системного развития. Нужен аккаунт GitHub (бесплатно), в котором хранится твоя личная база знаний, включая стратегию и проекты.

💡 Открой второй экран с claude.ai — спрашивай там всё непонятное в процессе установки.
  1. 1

    Создай аккаунт GitHub

    Если нет — зарегистрируйся на github.com (бесплатно). Там будет храниться твоя личная база знаний.

  2. 2

    Следуй полной инструкции

    Открой инструкцию на GitHub — от установки до первого запуска. Занимает около 15 минут.

  3. 3

    Запусти IWE

    После установки — первый запуск и полный доступ к автономному агенту, памяти и всем возможностям IWE.
