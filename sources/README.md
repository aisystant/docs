# Папка sources

Эта папка предназначена для работы с исходными материалами.

## Структура

- **word-files/** - размещайте Word-документы (`.docx`) для конвертации
- **format-guide.md** - руководство по форматированию Markdown

## Процесс работы

1. **Загрузка документа**: Поместите Word-файл в папку `word-files/`
2. **Коммит и PR**: Изменения фиксируются и отправляются в репозиторий
3. **Конвертация**: Word-документ конвертируется в Markdown согласно format-guide.md
4. **Размещение**: Готовые MD-файлы размещаются в соответствующих разделах docs/

## Примечания

- Используйте осмысленные имена файлов
- Убедитесь, что документ отформатирован в Word (заголовки, списки, таблицы)
- Изображения будут автоматически извлечены
