# Моделирование 5.2. Аннотирование типами с использованием нотации ::


Ознакомьтесь с примером в первой строке таблицы. Используйте наиболее удобный вариант разметки типов в тексте --- двойное двоеточие «::». Сначала пишется название объекта, затем его тип из мета-модели (или в добавок еще и мета-мета-модели). Например: «самолет::система» или «стол::система::физический объект». При необходимости повторите теоретический материал по нотации :: в руководстве «Рациональная работа».

Для предложений из таблички сделайте дубль, указав типы всех объектов и отношений, как показано в первой строке. Заполните пустые ячейки и напишите несколько примеров из рабочих и личных проектов.
