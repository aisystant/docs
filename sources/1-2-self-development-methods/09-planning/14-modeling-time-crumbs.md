# Моделирование 9.5. Список работ для крошек времени

Крошки времени — это небольшие промежутки свободного времени (5-15 минут), которые часто теряются в течение дня. Это может быть ожидание в очереди, поездка в транспорте, перерыв между встречами и т.п.

Для эффективного использования крошек времени необходимо заранее иметь список небольших задач, которые можно выполнить в эти промежутки. Составьте свой список работ для крошек времени:

| № | Работа | Примерное время | Категория | Примечания |
|---|--------|-----------------|-----------|------------|
| 1 | Пример: Прочитать статью по теме X | 10 мин | Обучение | Сохранить ссылку заранее |
| 2 | Пример: Ответить на сообщения в чате | 5 мин | Коммуникация | |
| 3 | Пример: Послушать подкаст | 15 мин | Развитие | Скачать заранее |
| 4 | Пример: Проговорить понятия изучаемой темы | 5 мин | Обучение | |
| 5 | Пример: Записать мимолетную заметку | 5 мин | Мышление письмом | |
| | | | | |
