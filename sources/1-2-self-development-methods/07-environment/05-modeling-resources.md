# Моделирование 7.1. Список информационных ресурсов. Формирование окружения

Составьте список информационных ресурсов (каналов), которые сейчас находятся в вашем окружении или которые вы намерены включить в свое окружение, чтобы постоянно пропитываться, мотивироваться и получать знания по практике формирование окружения. Необязательно иметь множество каналов, вполне достаточно если это будет руководство и клуб, но может у вас есть и другие. Например, каналы в телеграм по каким-то интересным для вас темам.
